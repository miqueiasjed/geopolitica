# Task 29.4 – AdminImportarAddons.tsx (upload + preview + template)

## Objetivo
Criar a página `AdminImportarAddons.tsx` com upload de arquivo CSV/XLSX, preview das primeiras 5 linhas após seleção e link para baixar o template CSV.

## Tipo
frontend-pagina

## Arquivos a criar/modificar
- `frontend/src/pages/admin/AdminImportarAddons.tsx` → criar

## Contexto necessário (arquivos para ler antes)
- `frontend/src/pages/admin/` → ver padrão de páginas admin
- `frontend/src/services/adminProdutos.ts` → `importarAddons()` e `exportarAddons()` (Task 28.2)

## Skills necessárias
- `frontend-design-system`
- `animacoes-motion`

## Especificação

### O que fazer:
**Seção de upload:**
- Dropzone ou input `accept=".csv,.xlsx,.xls"` com drag & drop visual
- Após seleção: exibir preview (5 primeiras linhas parsadas com PapaParse para CSV)
- Para XLSX: pode usar mensagem "Prévia disponível apenas para CSV"
- Botão "Importar" habilitado após seleção

**Template CSV:**
- Link "Baixar template" que gera/baixa CSV com cabeçalhos:
  `email,addon_key,status,fonte,iniciado_em,expira_em`
- Implementar como Blob download no frontend (sem request ao backend)

**Botão "Exportar addons atuais":**
- Chamar `adminProdutos.exportarAddons()`
- Fazer download via `URL.createObjectURL(blob)` com nome de arquivo `addons_ativos.csv`

**Estado durante importação:**
- Loading spinner no botão "Importar"
- Desabilitar seleção de novo arquivo durante importação

### Regras de negócio aplicáveis:
- Formatos aceitos: `.csv`, `.xlsx`, `.xls`
- Preview máximo de 5 linhas (não processar arquivo inteiro no frontend)
- Template gerado localmente para não depender do backend

### Critérios de aceitação:
- [ ] Dropzone aceita CSV e XLSX
- [ ] Preview de 5 linhas exibido para CSV após seleção
- [ ] Link "Baixar template" gera CSV com cabeçalhos corretos
- [ ] Botão "Exportar addons atuais" faz download do CSV
- [ ] Botão "Importar" fica em loading durante a operação

## Teste esperado
- Verificação visual: selecionar CSV de teste, confirmar preview
- Verificação: baixar template, confirmar cabeçalhos corretos
- Verificação: exportar addons, confirmar download

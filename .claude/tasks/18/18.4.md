# Task 18.4 – AdminAiUsoController + endpoint GET /api/admin/ai/uso + rota

## Objetivo
Implementar o endpoint `GET /api/admin/ai/uso` que retorna estatísticas agregadas de consumo de IA: resumo de hoje, mês atual, histórico dos últimos 7 dias e breakdown por serviço.

## Tipo
backend-endpoint

- `app/Http/Controllers/Api/Admin/AdminAiUsoController.php` → criar
- `routes/api.php` → modificar (adicionar rota)

- `app/Http/Controllers/Api/Admin/AdminConfiguracaoController.php` → padrão de controller admin

### Implementação:

```php
public function index(): JsonResponse
{
    $agora = now()->timezone('America/Sao_Paulo');
    $inicioDia = $agora->copy()->startOfDay();
    $inicioMes = $agora->copy()->startOfMonth();
    $inicio7Dias = $agora->copy()->subDays(6)->startOfDay();

    $hoje = AiLog::where('created_at', '>=', $inicioDia)
        ->selectRaw('COUNT(*) as chamadas, SUM(tokens_entrada) as tokens_entrada, SUM(tokens_saida) as tokens_saida, SUM(custo_estimado_usd) as custo_estimado_usd, AVG(duracao_ms) as duracao_media_ms')
        ->first();

    $mes = AiLog::where('created_at', '>=', $inicioMes)
        ->selectRaw('COUNT(*) as chamadas, SUM(tokens_entrada) as tokens_entrada, SUM(tokens_saida) as tokens_saida, SUM(custo_estimado_usd) as custo_estimado_usd, AVG(duracao_ms) as duracao_media_ms')
        ->first();

    $historico = AiLog::where('created_at', '>=', $inicio7Dias)
        ->selectRaw("DATE(CONVERT_TZ(created_at, '+00:00', '-03:00')) as data, COUNT(*) as chamadas, SUM(custo_estimado_usd) as custo_estimado_usd")
        ->groupBy('data')
        ->orderBy('data')
        ->get();

    $porServico = AiLog::where('created_at', '>=', $inicioMes)
        ->selectRaw('servico, COUNT(*) as chamadas, SUM(custo_estimado_usd) as custo_estimado_usd')
        ->groupBy('servico')
        ->orderByDesc('chamadas')
        ->get();

    return response()->json(['data' => [
        'hoje'            => $this->formatarResumo($hoje),
        'mes'             => $this->formatarResumo($mes),
        'historico_7_dias'=> $historico,
        'por_servico'     => $porServico,
        'provider_ativo'  => config('ai.provider', 'claude'),
        'modelo_ativo'    => $this->getModeloAtivo(),
        'threshold_alerta_usd' => (float) config('ai.threshold_alerta_usd', 10.0),
    ]]);
}
```

Helper privado `formatarResumo`: converte valores null para 0 e arredonda custo para 6 casas decimais.
Helper privado `getModeloAtivo`: retorna `config('claude.model')` se provider claude, `config('ai.openai.model')` se openai.

### Rota (dentro do grupo admin):
```php
Route::get('ai/uso', [AdminAiUsoController::class, 'index']);
```

### Regras de negócio aplicáveis:
- Datas e agrupamentos devem usar fuso `America/Sao_Paulo`.
- `threshold_alerta_usd` lido de `config('ai.threshold_alerta_usd', 10.0)` — o frontend usa para exibir aviso.
- Sem paginação — sempre retorna todos os dados agregados.

### Critérios de aceitação:
- [ ] `GET /api/admin/ai/uso` retorna 200 com estrutura completa.
- [ ] `hoje.chamadas` e `mes.chamadas` são inteiros ≥ 0.
- [ ] `historico_7_dias` tem até 7 entradas (uma por dia).
- [ ] `por_servico` lista apenas serviços que tiveram chamadas no mês atual.
- [ ] Sem token admin: resposta 403.
- [ ] Com banco vazio (sem logs): todos os campos numéricos retornam 0.

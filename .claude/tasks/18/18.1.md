### Migration:

```php
Schema::create('ai_logs', function (Blueprint $table) {
    $table->id();
    $table->string('provider', 20);           // 'claude', 'openai'
    $table->string('modelo', 60);             // ex: 'claude-sonnet-4-6', 'gpt-4o'

    $table->unsignedInteger('tokens_saida')->default(0);

    $table->unsignedInteger('duracao_ms')->default(0);
    $table->boolean('sucesso')->default(true);
    $table->text('erro')->nullable();

```

### Model AiLog:
```php
class AiLog extends Model
{
    public $timestamps = false; // só created_at, gerenciado pela migration
    const CREATED_AT = 'created_at';
    const UPDATED_AT = null;

    protected $fillable = [
        'provider', 'modelo', 'servico',
        'tokens_entrada', 'tokens_saida', 'custo_estimado_usd',
        'duracao_ms', 'sucesso', 'erro',
    ];

    protected $casts = [
        'sucesso' => 'boolean',
        'custo_estimado_usd' => 'float',
        'created_at' => 'datetime',
    ];

    // Preços por provider (USD por 1M tokens)
    private const PRECOS = [
        'claude' => ['entrada' => 3.00,  'saida' => 15.00],
        'openai' => [
            'gpt-4o'      => ['entrada' => 2.50, 'saida' => 10.00],
            'gpt-4o-mini' => ['entrada' => 0.15, 'saida' => 0.60],
        ],
    ];

    public static function calcularCusto(string $provider, string $modelo, int $tokensEntrada, int $tokensSaida): float
    {
        $precos = self::PRECOS[$provider] ?? null;
        if (!$precos) return 0.0;

        if ($provider === 'openai') {
            $precos = $precos[$modelo] ?? $precos['gpt-4o'];
        }

        return round(
            ($tokensEntrada / 1_000_000 * $precos['entrada']) +
            ($tokensSaida  / 1_000_000 * $precos['saida']),
            6
        );
    }
}
```

### Regras de negócio aplicáveis:
- Logs são imutáveis: sem `updated_at`.
- `custo_estimado_usd` deve ser calculado com `calcularCusto()` antes de criar o registro.
- Para provider desconhecido, custo é 0.

### Critérios de aceitação:
- [ ] `php artisan migrate` executa sem erros.
- [ ] Tabela `ai_logs` existe com todos os campos especificados.
- [ ] `AiLog::calcularCusto('claude', 'claude-sonnet-4-6', 1000, 500)` retorna valor > 0.
- [ ] `AiLog::calcularCusto('openai', 'gpt-4o-mini', 1000, 500)` retorna valor menor que com gpt-4o.
- [ ] `AiLog::calcularCusto('desconhecido', 'x', 100, 100)` retorna 0.

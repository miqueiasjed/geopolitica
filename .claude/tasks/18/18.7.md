# Task 18.7 – Gráfico de barras 7 dias CSS puro + badge provider ativo

## Objetivo
Criar o componente `AiUsoGrafico` que exibe o histórico dos últimos 7 dias como gráfico de barras em CSS puro (sem biblioteca de charts), e o badge de provider ativo com comparativo de custo Claude vs OpenAI.

## Tipo
frontend-componente

## Arquivos a criar/modificar
- `frontend/src/components/admin/AiUsoGrafico.tsx` → criar
- `frontend/src/pages/admin/ia/UsoPage.tsx` → modificar (integrar o componente)

- `frontend/src/types/ai.ts` → tipos AiUsoHistoricoDia e AiUsoData (Task 18.5)
- `frontend/src/pages/admin/ia/UsoPage.tsx` → onde integrar o gráfico (Task 18.6)

## Especificação

### Componente AiUsoGrafico:

**Props:**
```typescript
interface AiUsoGraficoProps {
  historico: AiUsoHistoricoDia[];    // array com até 7 itens
  providerAtivo: string;             // 'claude' | 'openai'
  modeloAtivo: string;
}
```

**Gráfico de barras CSS puro:**
- Container com `display: flex; align-items: flex-end; gap: 8px; height: 120px;`
- Cada barra: `<div>` com `width` fixo, `height` calculado como `(custo / maxCusto) * 100%`
- Se `maxCusto === 0`, todas as barras com altura mínima (4px para indicar "sem dados"
- Label de data abaixo de cada barra: dia/mês curto (ex: "21/04")
- Tooltip: ao hover, exibir `{data}: {chamadas} chamadas · $custo`
- Cor das barras: `bg-blue-500` para Claude, `bg-green-500` para OpenAI (basear na cor do provider ativo)

**Garantir 7 barras mesmo se o histórico tiver menos dias:**
Antes de renderizar, completar o array com dias faltantes (custo=0, chamadas=0) para os últimos 7 dias a partir de hoje.

**Badge de provider ativo:**
Renderizar acima ou ao lado do gráfico:
```
[Badge] Claude Sonnet 4.6   (badge azul se claude, verde se openai)
Preço: $3/M in · $15/M out
```

Mostrar comparativo simplificado:
- Claude Sonnet: $3/M entrada, $15/M saída
- GPT-4o: $2.50/M entrada, $10/M saída  
- GPT-4o-mini: $0.15/M entrada, $0.60/M saída

**Integração em UsoPage:**
Adicionar `<AiUsoGrafico historico={data.historico_7_dias} providerAtivo={data.provider_ativo} modeloAtivo={data.modelo_ativo} />` entre os cards de mês e a tabela de breakdown.

### Regras de negócio aplicáveis:
- Gráfico em CSS puro: proibido usar Recharts, Chart.js ou qualquer lib de gráficos.
- Valores monetários no tooltip: mesma formatação da Task 18.6 (6/4/2 casas decimais).
- Se `historico` vier vazio, exibir 7 barras de altura mínima com mensagem "Sem dados".

### Critérios de aceitação:
- [ ] Gráfico exibe exatamente 7 barras (completando dias sem dados com altura mínima).
- [ ] Altura das barras é proporcional ao custo do dia.
- [ ] Badge exibe provider ativo e modelo com preços de referência.
- [ ] Nenhuma biblioteca de gráficos importada.
- [ ] TypeScript compila sem erros.
- [ ] Responsivo: funciona em telas de 1024px e 1440px.

## Teste esperado

- Confirmar gráfico de 7 barras com proporções corretas.
- Confirmar badge com provider e preço.

# Task 4.2 – SourcesSeeder e RssFetcherService

## Objetivo
Popular o banco com as 8 fontes RSS iniciais e implementar o `RssFetcherService` responsável por coletar itens RSS com filtro de 24h e deduplicação por `fonte_url`.

## Tipo
backend-logica

## Arquivos a criar/modificar
- `database/seeders/SourcesSeeder.php` → criar
- `app/Services/RssFetcherService.php` → criar
- `database/seeders/DatabaseSeeder.php` → modificar (chamar SourcesSeeder)

## Contexto necessário (arquivos para ler antes)
- `app/Models/Source.php` → ver campos do Model (task 4.1)
- `app/Models/Event.php` → ver `fonte_url` para deduplicação (task 4.1)
- `config/feed.php` → ver configuração de fontes e intervalo (task 1.2)

## Skills necessárias
- laravel-arquitetura

## Especificação

### O que fazer:
1. Criar `SourcesSeeder` com as 8 fontes RSS:
   ```php
   ['nome' => 'Reuters World', 'rss_url' => 'https://feeds.reuters.com/reuters/worldNews', 'categoria' => 'geopolitica'],
   ['nome' => 'Al Jazeera English', 'rss_url' => 'https://www.aljazeera.com/xml/rss/all.xml', 'categoria' => 'geopolitica'],
   ['nome' => 'BBC World', 'rss_url' => 'http://feeds.bbci.co.uk/news/world/rss.xml', 'categoria' => 'geopolitica'],
   ['nome' => 'Financial Times', 'rss_url' => 'https://www.ft.com/rss/home/uk', 'categoria' => 'economia'],
   ['nome' => 'Bloomberg Markets', 'rss_url' => 'https://feeds.bloomberg.com/markets/news.rss', 'categoria' => 'economia'],
   ['nome' => 'Foreign Policy', 'rss_url' => 'https://foreignpolicy.com/feed/', 'categoria' => 'geopolitica'],
   ['nome' => 'The Economist', 'rss_url' => 'https://www.economist.com/international/rss.xml', 'categoria' => 'economia'],
   ['nome' => 'CNBC World', 'rss_url' => 'https://www.cnbc.com/id/100727362/device/rss/rss.html', 'categoria' => 'economia'],
   ```
   - Usar `firstOrCreate` com `rss_url` como chave
2. Criar `RssFetcherService`:
   - Método `fetchSource(Source $source): array` — retorna array de itens brutos
   - Usar `vknet/rss-reader` para fazer parse do feed
   - Filtrar itens com `pubDate` das últimas 24h
   - Filtrar itens cujo `link` já existe em `events.fonte_url` (deduplicação)
   - Atualizar `source->ultima_coleta_em = now()`
   - Retornar array de itens novos: `[['titulo', 'resumo', 'fonte_url', 'fonte', 'publicado_em'], ...]`
   - Em caso de erro de rede: logar com `Log::warning()` e retornar array vazio

### Regras de negócio aplicáveis:
- Deduplicação deve ocorrer **antes** de enviar para análise IA — economiza tokens
- Filtro de 24h evita reprocessar notícias antigas
- Erros de RSS de uma fonte não devem interromper a coleta das demais

### Critérios de aceitação:
- [ ] `php artisan db:seed --class=SourcesSeeder` insere as 8 fontes
- [ ] Executar duas vezes não duplica fontes (firstOrCreate)
- [ ] `RssFetcherService::fetchSource($source)` retorna array de itens sem erros de parse
- [ ] Itens com `fonte_url` já existente no banco não são retornados
- [ ] Itens com `pubDate` > 24h atrás são descartados

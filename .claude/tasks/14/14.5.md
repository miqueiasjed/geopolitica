## Especificação

### O que fazer:

**Instalar dependência (se não existir):**
```bash
npm install react-markdown
```

**`ChatInput.tsx`:**
Props: `onEnviar: (texto: string) => void`, `disabled: boolean`
- Textarea controlada (não input simples — suporta múltiplas linhas)
- Placeholder: "Faça uma pergunta sobre geopolítica..."
- Botão "Enviar" (ícone de seta) à direita
- `Ctrl+Enter` para enviar (não apenas Enter — Enter cria nova linha)
- Ao enviar: limpar textarea + chamar `onEnviar`
- `disabled` durante loading (streaming em andamento)
- Auto-resize da textarea (até 5 linhas)
- Contador de caracteres: "X / 500" no canto inferior direito

**`ChatMessage.tsx`:**
Props: `mensagem: ChatMensagem`
- `role === 'user'`: bolha à direita, fundo `#2A2F3A`, texto claro
- `role === 'assistant'`: bolha à esquerda, fundo `#1C1F26`, borda sutil dourada `#C9B882/30`
- Conteúdo renderizado com `ReactMarkdown` (suporte a **bold**, *italic*, listas, `código`)
- Se `mensagem.streaming === true`: exibir cursor piscante `▋` ao final do texto
- Avatar/ícone: user (ícone pessoa) | assistente (ícone globo/IA)
- Timestamp formatado como "14:32" se `created_at` presente

**`ChatMessages.tsx`:**
Props: `mensagens: ChatMensagem[]`, `isLoading: boolean`
- Lista vertical de `ChatMessage`
- Auto-scroll para o fundo quando nova mensagem é adicionada (`useEffect` + `ref.scrollIntoView`)
- Empty state: "Faça sua primeira pergunta sobre geopolítica"
- Skeleton de 2 mensagens durante carregamento inicial do histórico

**`ChatLimitWarning.tsx`:**
Props: `mensagem: string`, `onUpgrade?: () => void`
- Banner de aviso no topo da área de chat
- Texto do limite + CTA "Ver planos" (link para `/planos` ou modal de upgrade)
- Cor de fundo: âmbar `bg-amber-900/20`, borda `border-amber-700`

**`ChatBriefings.tsx` (página):**
Layout de duas colunas em desktop / single column em mobile:
- Coluna principal (70%): `ChatMessages` + `ChatInput`
- Coluna lateral (30%): badge "X / Y perguntas hoje" + dicas de perguntas

```tsx
const { mensagens, isLoading, limiteAtingido, limiteErro,
        perguntaCount, limite, enviar, carregarHistorico } = useChat();

useEffect(() => { carregarHistorico(); }, []);
```

- Se `limiteAtingido`: exibir `ChatLimitWarning`, desabilitar `ChatInput`
- Título: "Chat com os Briefings"
- Subtítulo: "Pergunte sobre qualquer evento, país ou crise geopolítica"
- Badge de contagem: `{perguntaCount} / {limite ?? '∞'} perguntas hoje`

**`TopNav.tsx`:**
- Adicionar link "Chat" com ícone de balão de fala
- Rota: `/dashboard/chat`

**Rota:**
```tsx
{ path: '/dashboard/chat', element: <ChatBriefings /> }
```

### Regras de negócio aplicáveis:
- `Ctrl+Enter` envia, `Enter` simples cria nova linha
- Streaming visível (tokens aparecem progressivamente)
- Auto-scroll ao fundo em cada novo token
- Desabilitar input enquanto streaming em andamento

### Critérios de aceitação:
- [ ] Mensagem do usuário aparece imediatamente ao enviar
- [ ] Resposta da IA aparece token por token com cursor piscante
- [ ] Auto-scroll funciona durante streaming
- [ ] `Ctrl+Enter` envia a mensagem
- [ ] Histórico do dia é carregado ao abrir a página
- [ ] `ChatLimitWarning` aparece e input fica disabled ao atingir limite
- [ ] Markdown renderizado corretamente nas respostas da IA
- [ ] Contador "X / Y perguntas hoje" atualiza após cada envio

**`src/types/chat.ts`:**

export type RoleMensagem = 'user' | 'assistant';

export interface ChatMensagem {

  limite: number | null; // null = sem limite

**`src/services/chatApi.ts`:**

`enviarPergunta` usa Fetch API com `ReadableStream`:
```typescript
const response = await fetch('/api/chat/perguntar', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${token}`,
    'Accept': 'text/event-stream',
  },
  body: JSON.stringify({ pergunta }),
});

if (response.status === 429) {
  const erro: ChatLimiteErro = await response.json();
  throw Object.assign(new Error(erro.message), { upgrade: true });
}

const reader = response.body!.getReader();
const decoder = new TextDecoder();

while (true) {
  const { done, value } = await reader.read();
  if (done) break;
  const chunk = decoder.decode(value);
  // parsear linhas SSE "data: {...}\n\n"
  for (const line of chunk.split('\n')) {
    if (line.startsWith('data: ')) {
      const data = line.slice(6);
      if (data === '[DONE]') return;
      const { token } = JSON.parse(data);
      yield token;
    }
  }
}
```

`carregarHistorico()`: chama `buscarHistorico()`, popula `mensagens`, `perguntaCount`, `limite`

`enviar(pergunta: string)`:
1. Adicionar mensagem do usuário (`role: 'user'`) ao state
2. Adicionar mensagem placeholder do assistente (`role: 'assistant', conteudo: '', streaming: true`)
3. Iterar sobre `enviarPergunta(pergunta, token)`:
   - Concatenar cada token ao `conteudo` da última mensagem
   - `setMensagens([...prev])` para trigger re-render
4. Ao terminar: setar `streaming: false` na última mensagem
5. Se erro 429: `setLimiteAtingido(true)`, `setLimiteErro(error.message)`
6. `setIsLoading(false)` no finally

Retornar: `{ mensagens, isLoading, limiteAtingido, limiteErro, perguntaCount, limite, enviar, carregarHistorico }`

### Regras de negócio aplicáveis:
- Token de autenticação obtido do contexto de auth já existente no projeto
- Mensagem de assistente em streaming deve atualizar progressivamente
- Erro 429 deve setar `limiteAtingido = true` para exibir `ChatLimitWarning`

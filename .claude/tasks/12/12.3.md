## Especificação

### O que fazer:

**`src/types/timeline.ts`:**
```typescript
export type CategoriaCrise =
  | 'guerra'
  | 'humanitária'
  | 'econômica'
  | 'financeira'
  | 'geopolítica'
  | 'terrorismo'
  | 'petróleo'
  | 'logística'
  | 'naval'
  | 'energética'
  | 'transição';

export interface MetricaTimeline {
  label: string;
  valor: string;
}

export interface CriseHistorica {
  id: number;
  titulo: string;
  slug: string;
  ano: number;
  data_inicio: string;
  data_fim: string | null; // null = em andamento
  categorias: CategoriaCrise[];
  content_slug: string | null;
}

export interface CriseHistoricaDetalhe extends CriseHistorica {
  contexto_geopolitico: string;
  impacto_global: string;
  impacto_brasil: string;
  metricas_globais: MetricaTimeline[];
  metricas_brasil: MetricaTimeline[];
}

export interface EventoTimeline {
  id: number;
  titulo: string;
  nivel_tensao: string;
  created_at: string;
}

export interface TimelineData {
  crises: CriseHistorica[];
  eventos: EventoTimeline[];
}

export interface FiltrosTimeline {
  periodo_inicio?: number;
  periodo_fim?: number;
  categoria?: CategoriaCrise;
}
```

**`useTimeline`:**
- Parâmetro: `filtros: FiltrosTimeline = {}`
- Query key: `['timeline', filtros]`
- Fetch: `GET /api/timeline` com filtros como query params
- `staleTime: 30 * 60 * 1000` (30 min — crises são estáticas, eventos ficam frescos)
- Retornar: `{ crises: CriseHistorica[], eventos: EventoTimeline[], isLoading, error }`

**`useCriseDetalhe`:**
- Parâmetro: `slug: string | null`
- Query key: `['crise-detalhe', slug]`
- Fetch: `GET /api/timeline/crise/{slug}`
- `enabled: !!slug`
- `staleTime: 24 * 60 * 60 * 1000` (24h — consistente com cache Redis)
- Retornar: `{ crise: CriseHistoricaDetalhe | undefined, isLoading, error }`

### Regras de negócio aplicáveis:
- `staleTime` de 24h para detalhe da crise (consistente com cache Redis backend)
- Hook de timeline deve repassar todos os filtros como query params
- `useCriseDetalhe` não faz fetch quando `slug` é null (painel fechado)

### Critérios de aceitação:
- [ ] Types TypeScript sem erros de compilação
- [ ] `useCriseDetalhe(null)` não dispara requisição
- [ ] `useTimeline({ categoria: 'guerra' })` inclui o filtro na URL
- [ ] `staleTime` corretos em ambos os hooks

## Objetivo

## Especificação

### O que fazer:

**Migration `crises_historicas`:**
```
id (bigIncrements)
titulo (string)
slug (string, unique)
ano (smallInteger) — ex: 1991
data_inicio (date)
data_fim (date, nullable) — null = crise em andamento
contexto_geopolitico (text)
impacto_global (text)
impacto_brasil (text)
metricas_globais (json, default []) — ex: [{"label": "Mortes estimadas", "valor": "800.000"}]
metricas_brasil (json, default []) — ex: [{"label": "Variação do câmbio", "valor": "+15%"}]
categorias (json, default []) — ex: ["guerra", "humanitária"]
content_slug (string, nullable) — slug de conteúdo na Biblioteca (M03)
timestamps
```

**Model `CriseHistorica`:**
- `$fillable`: todos os campos
- Casts: `data_inicio` e `data_fim` → `date`; `metricas_globais`, `metricas_brasil`, `categorias` → `array`
- `scopePorCategoria($query, string $categoria)`: `whereJsonContains('categorias', $categoria)`
- `scopePorPeriodo($query, int $anoInicio, int $anoFim)`: `whereBetween('ano', [$anoInicio, $anoFim])`
- Accessor `emAndamento()`: `return $this->data_fim === null`

**`CrisesHistoricasSeeder`:**
Usar `CriseHistorica::updateOrCreate(['slug' => $data['slug']], $data)` para idempotência.

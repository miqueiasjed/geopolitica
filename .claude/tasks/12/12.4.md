## Objetivo
Implementar o componente visual principal da Linha do Tempo: eixo horizontal navegável com scroll, faixa superior dourada para crises históricas e faixa inferior branca para eventos do M01.

## Especificação

### O que fazer:

**`CriseMarker.tsx`:**
Props: `crise: CriseHistorica`, `onClick: (slug: string) => void`, `posicaoX: number`
- Marcador dourado `#C9B882` na faixa superior
- Formato: linha vertical + círculo clicável no topo
- Tooltip ao hover: `crise.titulo` + `crise.ano`
- Largura do marcador proporcional à duração da crise (`data_fim - data_inicio`)
  - Se `data_fim` null: se estende até o ano atual
  - Cor com opacidade 60% na barra de duração, 100% no marcador inicial
- Ao clicar: chamar `onClick(crise.slug)`
- Cursor pointer

**`EventoMarker.tsx`:**
Props: `evento: EventoTimeline`, `onClick: (id: number) => void`, `posicaoX: number`
- Marcador branco `#E8E4DC` na faixa inferior
- Formato: losango ou círculo pequeno
- Tooltip ao hover: `evento.titulo` truncado (máx 40 chars)
- Ao clicar: chamar `onClick(evento.id)`
- Sem largura proporcional (ponto único)

**`TimelineBar.tsx`:**
Props: `crises: CriseHistorica[]`, `eventos: EventoTimeline[]`, `onCriseClick: (slug: string) => void`, `onEventoClick: (id: number) => void`

Layout SVG horizontal:
```
[Eixo de anos: 1990 ... 2024 ... hoje]
[Faixa superior dourada: CriseMarkers]
[Faixa inferior branca: EventoMarkers]
```

Cálculo de posição:
- `totalAnos = anoAtual - 1990 + 1`
- `larguraPorAno = svgWidth / totalAnos`
- `posicaoX(ano) = (ano - 1990) * larguraPorAno`

SVG:
- Largura: `3000px` (scroll horizontal via container overflow-x-auto)
- Altura: `200px`
- Faixa superior (y: 40-100): crises douradas `#C9B882`
- Faixa inferior (y: 110-160): eventos brancos `#E8E4DC`
- Linha central separadora: `y=105`, cor cinza sutil
- Eixo de anos: marcadores a cada 2 anos com label

Navegação:
- Container com `overflow-x-auto` e `scroll-snap`
- Botões "← Anterior" e "Próximo →" para scroll por décadas
- Highlight no ano atual com linha vertical pontilhada

Animação (Framer Motion):
- `initial={{ opacity: 0 }}` + `animate={{ opacity: 1 }}` no mount do SVG
- Hover nos markers com `scale: 1.3` via `whileHover`

### Regras de negócio aplicárias:
- Cor crises: `#C9B882` (dourado)
- Cor eventos: `#E8E4DC` (branco suave)
- Scroll horizontal via overflow, não paginação

### Critérios de aceitação:
- [ ] Eixo de anos de 1990 até o ano atual renderizado
- [ ] Marcadores dourados nas posições corretas para crises
- [ ] Marcadores brancos para eventos recentes na faixa inferior
- [ ] Scroll horizontal funcionando
- [ ] Tooltip aparece no hover
- [ ] `onCriseClick` chamado com o slug correto

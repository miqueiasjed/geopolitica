## Especificação

### O que fazer:

**Migration `eleicoes`:**
```
id (bigIncrements)
pais (string) — nome do país em pt-BR
codigo_pais (char 2) — ISO 3166-1 alpha-2
data_eleicao (date)
tipo_eleicao (string) — ex: "Presidencial", "Parlamentar", "Referendo"
relevancia (enum: 'alta', 'media', 'baixa')
contexto_geopolitico (text)
impacto_brasil (text)
candidatos_principais (json, default []) — ex: [{"nome": "John Doe", "partido": "Partido X"}]
content_slug (string, nullable) — link para Biblioteca M03
timestamps
```

**Model `Eleicao`:**
- `$fillable`: todos os campos
- Casts: `data_eleicao` → `date`; `candidatos_principais` → `array`
- `scopePorAno($query, int $ano)`: `whereYear('data_eleicao', $ano)`
- `scopePorRelevancia($query, string $relevancia)`: `where('relevancia', $relevancia)`
- `scopeOrdenadaPorData($query)`: `orderBy('data_eleicao', 'ASC')`

**`EleicoesSeedSeeder`:**
Usar `Eleicao::updateOrCreate(['pais' => $data['pais'], 'data_eleicao' => $data['data_eleicao']], $data)`.

# Task 13.4 – RadarGrid e EleicaoCard

## Objetivo
Implementar a grade visual mensal de eleições (12 colunas = 12 meses) e o card individual de eleição com badge de relevância colorido.

## Tipo
frontend-componente

## Arquivos a criar/modificar
- `src/components/eleicoes/RadarGrid.tsx` → criar
- `src/components/eleicoes/EleicaoCard.tsx` → criar

## Contexto necessário (arquivos para ler antes)
- `src/types/eleicao.ts` → Eleicao, CORES_RELEVANCIA (13.3)
- `src/hooks/useEleicoes.ts` → estrutura dos dados (13.3)
- `src/components/` → ver padrão de componentes existentes

## Skills necessárias
- frontend-design-system

## Especificação

### O que fazer:

**`EleicaoCard.tsx`:**
Props: `eleicao: Eleicao`, `onClick: (id: number) => void`

Conteúdo:
- Badge de relevância no canto superior direito (cor conforme `CORES_RELEVANCIA`)
  - `alta` → badge vermelho `#EF4444` com texto "Alta"
  - `media` → badge amarelo `#FACC15` com texto "Média"
  - `baixa` → badge branco translúcido com texto "Baixa"
- Bandeira emoji (buscar de um mapa estático `BANDEIRAS: Record<string, string>` baseado em `codigo_pais`)
- Nome do país em destaque
- Data formatada: `dd/MM/yyyy`
- Tipo de eleição em texto menor (ex: "Presidencial")
- Ao clicar: `onClick(eleicao.id)`
- Hover: borda da cor de relevância + `scale(1.02)` suave

**`RadarGrid.tsx`:**
Props: `eleicoes: Eleicao[]`, `onEleicaoClick: (id: number) => void`

Layout grade:
- 12 colunas = 12 meses (Janeiro a Dezembro)
- Header da grade: nome dos meses abreviados (Jan, Fev, Mar...)
- Cada coluna: lista vertical de `EleicaoCard` para eleições daquele mês
- Altura mínima por coluna: `120px` (mesmo sem eleições)
- Coluna vazia: placeholder sutil com texto "Sem eleições" em cinza

Lógica de agrupamento:
```typescript
const eleicoesPorMes = Array.from({ length: 12 }, (_, i) =>
  eleicoes.filter(e => new Date(e.data_eleicao).getMonth() === i)
);
```

Estilo da grade:
- Container com `overflow-x-auto` (scroll horizontal em telas pequenas)
- Largura mínima: `900px` (garante 12 colunas visíveis em desktop)
- Separadores verticais sutis entre meses (borda cinza `1px`)
- Header do mês com destaque leve (fundo ligeiramente diferente)
- Mês atual destacado com borda dourada `#C9B882`

### Regras de negócio aplicáveis:
- Cor do badge de relevância: `alta=#EF4444`, `media=#FACC15`, `baixa=rgba(232,228,220,0.2)`
- Grade organiza por mês da `data_eleicao`
- Scroll horizontal em telas menores que 900px

### Critérios de aceitação:
- [ ] 12 colunas renderizadas com nomes dos meses
- [ ] Eleições posicionadas no mês correto
- [ ] Badge de relevância com cor correta
- [ ] Coluna do mês atual com destaque dourado
- [ ] Hover no card com efeito visual
- [ ] `onClick` propaga o `id` correto

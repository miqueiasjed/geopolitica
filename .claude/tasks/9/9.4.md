## Especificação

### O que fazer:

1. **`src/types/indicadores.ts`**:
   ```ts
   export interface Indicador {
     id: number
     simbolo: string
     nome: string
     valor: number | null
     moeda: 'USD' | 'BRL'
     unidade: string
     variacao_pct: number | null
     variacao_abs: number | null
     atualizado_em: string | null
   }

   export interface IndicadorHistoricoItem {
     valor: number
     registrado_em: string
   }

   export interface IndicadorHistoricoResponse {
     simbolo: string
     historico: IndicadorHistoricoItem[]
   }
   ```

2. **`useIndicadores.ts`**:
   - `useQuery` com chave `['indicadores']`
   - Query function: `GET /api/indicadores`
   - `refetchInterval: 15 * 60 * 1000` (15 minutos)
   - Expõe: `indicadores: Indicador[]`, `isLoading`, `ultimaAtualizacao` (timestamp do primeiro item)

3. **`useIndicadorHistorico.ts`** (params: `simbolo: string`, `enabled?: boolean`):
   - `useQuery` com chave `['indicador-historico', simbolo]`
   - Query function: `GET /api/indicadores/historico?simbolo=${simbolo}`
   - `enabled`: só busca quando `enabled !== false`
   - Expõe: `historico: IndicadorHistoricoItem[]`, `isLoading`

4. **`IndicatorCard.tsx`** (props: `indicador: Indicador`, `historico?: IndicadorHistoricoItem[]`):
   - Nome do indicador (ex: "Petróleo Brent")
   - Valor formatado:
     - USD: `$ 85,42` (2 casas decimais)
     - BRL: `R$ 5,23` (2 casas decimais)
   - Variação em % colorida:
     - positivo → `text-green-400` com `▲`
     - negativo → `text-red-400` com `▼`
     - zero/null → `text-zinc-400`
   - Sparkline SVG inline (64×28px, sem biblioteca):
     - Normaliza valores do histórico para 0–28px de altura
     - Desenha `<polyline>` com pontos calculados
     - Cor da linha: verde se última variação positiva, vermelho se negativa
     - Se `historico` vazio ou undefined: não renderiza sparkline

### Regras de negócio aplicáveis:
- Sparkline sem biblioteca externa — apenas SVG `polyline` calculado inline
- Cores verde/vermelho refletem a variação atual (`variacao_pct`)
- `refetchInterval` de 15 minutos sincroniza com o cache do backend

### Critérios de aceitação:
- [ ] `useIndicadores` retorna 6 indicadores em ordem
- [ ] `IndicatorCard` renderiza valor formatado conforme moeda
- [ ] Variação positiva → texto verde com ▲; negativa → vermelho com ▼
- [ ] Sparkline SVG renderiza quando `historico` tem dados
- [ ] TypeScript compila sem erros

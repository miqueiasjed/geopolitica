# Task 9.3 – IndicadoresController, HistoricoController e Rotas

## Objetivo
Criar os dois controllers de leitura dos indicadores (valores atuais e histórico), o FormRequest de validação de símbolo e as rotas correspondentes.

## Tipo
backend-endpoint

## Arquivos a criar/modificar
- `app/Http/Controllers/Api/IndicadoresController.php` → criar
- `app/Http/Controllers/Api/IndicadoresHistoricoController.php` → criar
- `app/Http/Requests/IndicadorHistoricoRequest.php` → criar
- `routes/api.php` → modificar

## Contexto necessário (arquivos para ler antes)
- `app/Services/IndicadoresService.php` → assinatura de `listarComCache()` e `historicoPorSimbolo()`
- `routes/api.php` → ver estrutura de grupos de rotas existentes

## Skills necessárias
- laravel-arquitetura

## Especificação

### O que fazer:

1. **`IndicadoresController`**:
   - `index(): JsonResponse`
     - Chama `IndicadoresService::listarComCache()`
     - Retorna `response()->json(['data' => $indicadores])`
   - Middleware: `auth:sanctum`

2. **`IndicadoresHistoricoController`**:
   - `index(IndicadorHistoricoRequest $request): JsonResponse`
     - Recebe `simbolo` via query param
     - Chama `IndicadoresService::historicoPorSimbolo($request->simbolo)`
     - Retorna `response()->json(['simbolo' => $simbolo, 'historico' => $dados])`
   - Middleware: `auth:sanctum`

3. **`IndicadorHistoricoRequest`**:
   - `simbolo` (required, string, in:`BZ=F`,`USDBRL=X`,`NG=F`,`ZS=F`,`ZW=F`,`TIO=F`)
   - Mensagem de erro personalizada: "Símbolo inválido. Use um dos símbolos permitidos."

4. **Rotas em `routes/api.php`** (grupo `auth:sanctum`):
   ```php
   Route::get('/indicadores', [IndicadoresController::class, 'index']);
   Route::get('/indicadores/historico', [IndicadoresHistoricoController::class, 'index']);
   ```

### Regras de negócio aplicáveis:
- Controllers finos — apenas orquestração
- Símbolo deve ser validado explicitamente — lista fixa de 6 permitidos
- Cache é responsabilidade do Service, não do Controller

### Critérios de aceitação:
- [ ] `GET /api/indicadores` retorna 6 indicadores na ordem correta
- [ ] `GET /api/indicadores/historico?simbolo=BZ=F` retorna histórico de 7 dias
- [ ] `GET /api/indicadores/historico?simbolo=INVALIDO` retorna HTTP 422
- [ ] `GET /api/indicadores` sem token retorna HTTP 401
- [ ] `php artisan route:list | grep indicadores` mostra as 2 rotas

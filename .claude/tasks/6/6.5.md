# Task 6.5 – Componente WorldMap com escala de cores e interação

## Objetivo
Criar o componente `WorldMap` usando `react-simple-maps` e `d3-scale` com escala de cores por intensidade, hover dourado, tooltip e handler de clique que abre o painel regional.

## Tipo
frontend-componente

## Arquivos a criar/modificar
- `frontend/src/components/WorldMap.tsx` → criar
- `frontend/src/components/IntensityLegend.tsx` → criar
- `public/world-110m.json` → baixar (TopoJSON Natural Earth 110m)

## Contexto necessário (arquivos para ler antes)
- `frontend/src/types/mapa.ts` → ver `PaisIntensidade` (task 6.4)
- `frontend/src/hooks/useMapaIntensidade.ts` → ver dados disponíveis (task 6.4)

## Skills necessárias
- frontend-design-system
- animacoes-motion

## Especificação

### O que fazer:
1. Instalar dependências:
   ```
   npm install react-simple-maps d3-scale d3-scale-chromatic topojson-client
   npm install @radix-ui/react-tooltip
   ```
2. Baixar TopoJSON: `curl -o public/world-110m.json https://cdn.jsdelivr.net/npm/world-atlas@2/countries-110m.json`
3. Criar `WorldMap.tsx`:
   - Props: `paises: PaisIntensidade[]`, `onPaisClick: (codigo: string, nome: string) => void`
   - Usar `ComposableMap` + `Geographies` do `react-simple-maps`
   - Escala de cores: `scaleLinear` de d3, domínio `[1, 10]`, range `['#2a2a2a', '#7f1d1d']`
   - Para cada país: buscar `intensidade_final` pelo `ISO_A2` do TopoJSON
   - Hover: border dourado `#C9B882`, cursor pointer
   - Tooltip com `@radix-ui/react-tooltip`: nome do país + intensidade ao hover
   - Click: chamar `onPaisClick(codigo, nome)`
4. Criar `IntensityLegend.tsx`:
   - Gradiente visual de `#2a2a2a` → `#7f1d1d`
   - Labels: "Baixo (1)" → "Alto (10)"
   - Posição: `fixed bottom-6 left-6`

### Regras de negócio aplicáveis:
- Países sem dados no GDELT ficam com cor `#2a2a2a` (intensidade mínima)
- Hover cor dourada é temporária — ao sair do hover, volta à cor de intensidade
- O mapa deve ter `projectionConfig` com zoom e centro configurados para visualização global

### Critérios de aceitação:
- [ ] Mapa renderiza com países coloridos por intensidade
- [ ] País com `intensidade_final = 10` aparece em vermelho escuro `#7f1d1d`
- [ ] País sem dados aparece em `#2a2a2a`
- [ ] Hover muda border/cor para dourado
- [ ] Tooltip mostra nome e intensidade
- [ ] Click dispara `onPaisClick` com código e nome corretos
- [ ] `IntensityLegend` visível no canto inferior esquerdo

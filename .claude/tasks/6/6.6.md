# Task 6.6 – RegionPanel animado e página Mapa

## Objetivo
Criar o `RegionPanel` com animação Framer Motion que exibe eventos do país clicado, e a página `Mapa.tsx` que compõe o mapa com o painel. Atualizar o TopNav com o link "Mapa de Calor".

## Tipo
frontend-pagina

## Arquivos a criar/modificar
- `frontend/src/components/RegionPanel.tsx` → criar
- `frontend/src/pages/Mapa.tsx` → criar
- `frontend/src/components/TopNav.tsx` → modificar (adicionar link Mapa)
- `frontend/src/App.tsx` → modificar (adicionar rota /dashboard/mapa)

## Contexto necessário (arquivos para ler antes)
- `frontend/src/hooks/useRegiaoEventos.ts` → ver interface (task 6.4)

## Especificação

### O que fazer:
1. Criar `RegionPanel.tsx`:
   - Props: `regiao: string | null`, `nome: string | null`, `onClose: () => void`
   - Usar `AnimatePresence` do Framer Motion
   - Quando `regiao !== null`:
     - Painel desliza da direita: `initial={{ x: 420 }}`, `animate={{ x: 0 }}`, `exit={{ x: 420 }}`
     - Largura: 420px, posição `fixed right-0 top-0 h-full`
     - Fundo: `bg-[#0d0d0f] border-l border-[#1e1e20]`
     - Header: nome do país + botão fechar (X)
     - Corpo: usar `useRegiaoEventos(regiao)` para buscar eventos
     - Listar eventos com versão simplificada do EventCard (sem análise IA)
     - `LoadingSpinner` durante carregamento
2. Criar `Mapa.tsx`:
   - Estado: `{ codigoPais: string | null, nomePais: string | null }`
   - Renderizar `WorldMap` + `RegionPanel` + `IntensityLegend`
   - `onPaisClick`: setar estado com código e nome
   - `onClose`: setar estado para null (fecha painel com animação de saída)
3. Atualizar `TopNav.tsx`:
   - Adicionar link "Mapa de Calor" → `/dashboard/mapa`
   - Link ativo: cor dourada `#C9B882`
4. Atualizar `App.tsx`:
   ```tsx
   <Route path="/dashboard/mapa" element={<Mapa />} />
   ```

### Regras de negócio aplicáveis:
- O painel deve fechar suavemente com `AnimatePresence` (exit animation)
- Clicar fora do painel (no mapa) pode fechar o painel — adicionar overlay ou deixar livre
- O mapa não deve ter scroll — usar `h-screen overflow-hidden` no container

### Critérios de aceitação:
- [ ] Clicar em país no mapa abre `RegionPanel` com animação
- [ ] Painel exibe eventos do país com `ImpactBadge`
- [ ] Botão X fecha o painel com animação de saída
- [ ] `AnimatePresence` funciona (exit animation visível)
- [ ] TopNav tem link "Mapa de Calor" ativo na rota `/dashboard/mapa`
- [ ] Mapa ocupa tela completa sem scroll vertical

# Task 6.3 – MapaIntensidadeController e RegiaoEventosController

## Objetivo
Implementar os dois controllers do mapa com os endpoints de intensidade por país (com cache) e eventos por região, registrar as rotas protegidas.

## Tipo
backend-endpoint

## Arquivos a criar/modificar
- `app/Http/Controllers/MapaIntensidadeController.php` → criar
- `app/Http/Controllers/RegiaoEventosController.php` → criar
- `routes/api.php` → modificar (adicionar rotas mapa)

## Contexto necessário (arquivos para ler antes)
- `app/Services/MapaIntensidadeService.php` → ver interface (task 6.2)
- `app/Models/Event.php` → ver scope `porRegiao` e `ultimas48h` (task 4.1)
- `routes/api.php` → ver estrutura atual

## Skills necessárias
- laravel-arquitetura

## Especificação

### O que fazer:
1. Criar `MapaIntensidadeController::index()`:
   - Chamar `MapaIntensidadeService::obter()`
   - Retornar array de `{ codigo_pais, nome_pais, intensidade_final }`
   - Cache da response HTTP: `response()->header('Cache-Control', 'public, max-age=900')`
2. Criar `RegiaoEventosController::index(Request $request)`:
   - Validar `regiao` (required, string)
   - Buscar `Event::relevantes()->porRegiao($regiao)->ultimas48h()->orderBy('impact_score', 'desc')->limit(10)->get()`
   - Retornar `EventResource::collection($events)`
3. Registrar rotas:
   ```php
   Route::middleware(['auth:sanctum', 'assinante.ativo'])->prefix('mapa')->group(function () {
       Route::get('/intensidade', [MapaIntensidadeController::class, 'index']);
       Route::get('/regiao-eventos', [RegiaoEventosController::class, 'index']);
   });
   ```

### Regras de negócio aplicáveis:
- Todos os assinantes têm acesso ao mapa (sem restrição por plano neste endpoint)
- O endpoint `/mapa/intensidade` deve ter cache HTTP além do Redis para reduzir carga
- `/mapa/regiao-eventos` retorna no máximo 10 eventos — suficiente para o painel lateral

### Critérios de aceitação:
- [ ] `GET /api/mapa/intensidade` retorna array de países com `intensidade_final`
- [ ] `GET /api/mapa/regiao-eventos?regiao=Europa` retorna eventos da região
- [ ] Ambos endpoints retornam 401 sem autenticação
- [ ] Resposta de intensidade tem header `Cache-Control: public, max-age=900`
- [ ] Parâmetro `regiao` ausente em `/regiao-eventos` retorna 422

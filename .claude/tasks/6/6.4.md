# Task 6.4 – Frontend: types, hooks useMapaIntensidade e useRegiaoEventos

## Objetivo
Criar as interfaces TypeScript do mapa, o hook `useMapaIntensidade` com refetch a cada 10 minutos e o hook `useRegiaoEventos` para carregar eventos ao clicar num país.

## Tipo
frontend-componente

- `frontend/src/hooks/useMapaIntensidade.ts` → criar
- `frontend/src/hooks/useRegiaoEventos.ts` → criar

## Especificação

### O que fazer:
1. Criar `src/types/mapa.ts`:
   ```ts
   export interface PaisIntensidade {
     codigo_pais: string      // ISO alpha-2: "BR", "US"
     nome_pais: string
     intensidade_final: number // 1–10
   }
   export interface MapaData {
     paises: PaisIntensidade[]
   }
   ```
2. Criar `useMapaIntensidade.ts`:
   - `useQuery({ queryKey: ['mapa-intensidade'], queryFn: () => api.get('/mapa/intensidade') })`
   - `refetchInterval: 10 * 60 * 1000` (10 minutos)
   - `staleTime: 5 * 60 * 1000`
   - Retornar `{ paises, isLoading, error }`
3. Criar `useRegiaoEventos.ts`:
   - `useQuery({ queryKey: ['regiao-eventos', regiao], queryFn: () => api.get('/mapa/regiao-eventos', { params: { regiao } }), enabled: !!regiao })`
   - `staleTime: 2 * 60 * 1000`
   - Retornar `{ eventos, isLoading }`

### Regras de negócio aplicáveis:
- `useRegiaoEventos` só busca quando `regiao` está definida (`enabled: !!regiao`)
- O mapa atualiza a cada 10 minutos (mais lento que o feed de 5 min)
- `staleTime` configurado para reduzir refetches desnecessários

### Critérios de aceitação:
- [ ] `useMapaIntensidade()` retorna array de países com `intensidade_final`
- [ ] `useRegiaoEventos('Europa')` retorna eventos da Europa
- [ ] `useRegiaoEventos(null)` não faz request (enabled: false)
- [ ] Refetch automático a cada 10 minutos para intensidade
- [ ] TypeScript não tem erros de tipo

## Especificação

### O que fazer:

**Migration `empresas`:**
```
id, nome (string), subdominio (string, unique),
logo_url (string, nullable), ativo (boolean, default true),
max_usuarios (unsignedSmallInteger, default 5),
expira_em (timestamp, nullable), timestamps
```

**Migration `licencas_b2b`:**
```
id, empresa_id (foreignId → empresas, cascadeOnDelete),
tipo (string, default 'standard'),
ativa (boolean, default true),
contratado_em (timestamp), expira_em (timestamp), timestamps
```

**Migration `membros_b2b`:**
```
id, empresa_id (foreignId → empresas, cascadeOnDelete),
user_id (foreignId → users, cascadeOnDelete, nullable) — null enquanto convite pendente,
role_b2b (enum: 'company_admin', 'reader'),
convite_token (string, unique, nullable),
convite_email (string, nullable),
aceito_em (timestamp, nullable),
timestamps
UNIQUE KEY (empresa_id, user_id) — previne membros duplicados
```

**Model `Empresa`:**
- `$fillable`: todos os campos
- Casts: `ativo` → `boolean`; `expira_em` → `datetime`
- `membros()`: hasMany `MembroB2B`
- `licenca()`: hasOne `LicencaB2B` → `latestOfMany()`
- `estaAtiva()`: `$this->ativo && ($this->expira_em === null || $this->expira_em->isFuture())`

**Model `LicencaB2B`:**
- `$fillable`: todos os campos
- Casts: `ativa` → `boolean`; `contratado_em`, `expira_em` → `datetime`
- `empresa()`: belongsTo `Empresa`

**Model `MembroB2B`:**
- `$fillable`: todos os campos
- Casts: `aceito_em` → `datetime`
- `empresa()`: belongsTo `Empresa`
- `usuario()`: belongsTo `User` (nullable — pode ser convite pendente)
- Scope `scopePendentes($query)`: `whereNull('aceito_em')`
- Scope `scopeAtivos($query)`: `whereNotNull('aceito_em')`

**`RolesB2BSeeder`:**
```php
Role::firstOrCreate(['name' => 'company_admin', 'guard_name' => 'web']);
Role::firstOrCreate(['name' => 'reader', 'guard_name' => 'web']);
```

**`IdentificarTenantMiddleware`:**
```php
public function handle(Request $request, Closure $next)
{
    $host = $request->getHost();
    $baseDomain = config('app.domain');
    
    // Remover o domínio base para obter o subdomínio
    $subdominio = str_replace('.' . $baseDomain, '', $host);
    
    // Se é o domínio raiz ou www, não é B2B
    if ($subdominio === $baseDomain || $subdominio === 'www') {
        return $next($request);
    }
    
    $empresa = Empresa::where('subdominio', $subdominio)
        ->where('ativo', true)
        ->first();
    
    if (!$empresa || !$empresa->estaAtiva()) {
        abort(404, 'Licença não encontrada ou expirada.');
    }
    
    // Injetar empresa no request para uso nos controllers
    $request->merge(['empresa_id' => $empresa->id]);
    app()->instance('tenant', $empresa);
    
    return $next($request);
}
```

Registrar em `Kernel.php` como middleware global ou de grupo `web`/`api`.

### Regras de negócio aplicáveis:
- `subdominio` é único — validado via `UNIQUE KEY`
- Middleware não aborta se for domínio raiz (usuários individuais)
- Convite pode existir sem `user_id` (pendente)

### Critérios de aceitação:
- [ ] 3 migrations executam sem erros
- [ ] `Empresa::estaAtiva()` retorna false para empresa expirada
- [ ] `RolesB2BSeeder` cria roles `company_admin` e `reader`
- [ ] `IdentificarTenantMiddleware` injeta `empresa_id` para subdomínio válido
- [ ] Middleware retorna 404 para subdomínio inativo

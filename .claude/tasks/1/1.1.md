# Task 1.1 – Criar projeto Laravel 13 e instalar dependências backend

## Objetivo
Criar o projeto Laravel 13 via `composer create-project`, configurar o `.env` completo e instalar todas as dependências necessárias (Sanctum, Spatie Permission, Anthropic SDK PHP, vknet/rss-reader).

## Tipo
backend-estrutura

## Arquivos a criar/modificar
- `composer.json` → modificar (após instalação das dependências)
- `.env` → criar (a partir do `.env.example`)
- `.env.example` → modificar (adicionar variáveis customizadas)
- `config/app.php` → modificar (registrar service providers se necessário)

## Contexto necessário (arquivos para ler antes)
- Nenhum — é o ponto de partida do projeto

## Skills necessárias
- laravel-arquitetura

## Especificação

### O que fazer:
1. Executar `composer create-project laravel/laravel backend "^13.0"` na raiz do repositório
2. Instalar dependências via composer:
   - `composer require laravel/sanctum`
   - `composer require spatie/laravel-permission`
   - `composer require anthropic-php/client` (ou SDK disponível)
   - `composer require vknet/rss-reader`
3. Publicar configs necessárias:
   - `php artisan vendor:publish --provider="Laravel\Sanctum\SanctumServiceProvider"`
   - `php artisan vendor:publish --provider="Spatie\Permission\PermissionServiceProvider"`
4. Configurar `.env` com as seguintes variáveis:
   - `DB_CONNECTION=mysql`, `DB_DATABASE`, `DB_USERNAME`, `DB_PASSWORD`
   - `REDIS_HOST`, `REDIS_PORT`
   - `CLAUDE_API_KEY=`
   - `SANCTUM_STATEFUL_DOMAINS=localhost:5173`
   - `MAIL_MAILER=resend`, `MAIL_FROM_ADDRESS`, `MAIL_FROM_NAME`
   - `HOTMART_WEBHOOK_TOKEN=`
5. Adicionar variáveis ao `.env.example` (sem valores sensíveis)

### Regras de negócio aplicáveis:
- O projeto Laravel deve residir em `backend/` na raiz do repositório
- Não há cadastro público — o `.env.example` deve documentar todas as variáveis necessárias

### Critérios de aceitação:
- [ ] `php artisan --version` retorna Laravel 13.x
- [ ] `composer show | grep sanctum` confirma instalação
- [ ] `composer show | grep spatie/laravel-permission` confirma instalação
- [ ] `.env` preenchido com todas as variáveis (sem valores sensíveis no `.env.example`)
- [ ] `php artisan config:clear` executa sem erros

## Teste esperado

- Verificação manual: checar se todos os packages aparecem em `vendor/`

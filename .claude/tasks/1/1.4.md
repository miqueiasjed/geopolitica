# Task 1.4 – Criar projeto React 19 + Vite + TypeScript com stack completa

## Objetivo
Criar o projeto frontend com React 19, Vite, TypeScript, e instalar todas as dependências necessárias: TailwindCSS 4, Radix UI, TanStack Query v5, React Router v7. Configurar estrutura de pastas padrão e cliente Axios com base URL.

## Tipo
frontend-pagina

## Arquivos a criar/modificar
- `frontend/` → criar (diretório do projeto Vite)
- `frontend/src/lib/axios.ts` → criar
- `frontend/src/main.tsx` → modificar (adicionar providers)
- `frontend/tailwind.config.ts` → criar/verificar
- `frontend/vite.config.ts` → modificar (proxy para API)

## Contexto necessário (arquivos para ler antes)
- Nenhum — é o ponto de partida do frontend

## Skills necessárias
- frontend-design-system

## Especificação

### O que fazer:
1. Executar `npm create vite@latest frontend -- --template react-ts` na raiz do repositório
2. Instalar dependências:
   ```
   npm install react-router-dom@^7
   npm install @tanstack/react-query@^5
   npm install axios
   npm install @radix-ui/themes @radix-ui/react-icons
   npm install tailwindcss @tailwindcss/vite
   npm install -D @types/node
   ```
3. Configurar `vite.config.ts` com proxy:
   ```ts
   server: {
     proxy: {
       '/api': 'http://localhost:8000'
     }
   }
   ```
4. Configurar TailwindCSS 4 no `vite.config.ts` e `src/index.css`
5. Criar `src/lib/axios.ts`:
   ```ts
   import axios from 'axios'
   const api = axios.create({ baseURL: '/api' })
   export default api
   ```
6. Criar estrutura de pastas:
   ```
   src/
     components/
     pages/
     hooks/
     lib/
     contexts/
     types/
   ```
7. Configurar `main.tsx` com `QueryClientProvider` e `BrowserRouter`

### Regras de negócio aplicáveis:
- Background padrão da aplicação: `#0a0a0b` (tema escuro)
- O proxy do Vite evita problemas de CORS em desenvolvimento
- Token JWT será armazenado em `localStorage` — o interceptor Axios deve incluir o header `Authorization`

### Critérios de aceitação:
- [ ] `npm run dev` inicia sem erros na porta 5173
- [ ] `npm run build` compila sem erros TypeScript
- [ ] `src/lib/axios.ts` exporta instância configurada com baseURL `/api`
- [ ] `main.tsx` tem `QueryClientProvider` e `BrowserRouter` envolvendo o app
- [ ] Estrutura de pastas `pages/`, `components/`, `hooks/`, `contexts/`, `lib/`, `types/` criada
- [ ] TailwindCSS funcional (classe `bg-[#0a0a0b]` aplicada no body)

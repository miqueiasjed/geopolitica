## Especificação

### O que fazer:

1. **Instalar dependências** (via npm):
   ```
   npm install @tiptap/react @tiptap/pm @tiptap/starter-kit @tiptap/extension-placeholder mark.js
   ```

2. **`src/types/biblioteca.ts`**:
   ```ts
   export type TipoConteudo = 'briefing' | 'mapa' | 'tese'
   export type PlanoMinimo = 'essencial' | 'pro' | 'reservado'

   export interface ConteudoCard {
     id: number
     tipo: TipoConteudo
     titulo: string
     slug: string
     resumo: string
     regiao: string | null
     tags: string[] | null
     tese_manchete: string | null
     publicado_em: string // ISO 8601
   }

   export interface Conteudo extends ConteudoCard {
     corpo: string // HTML
     plano_minimo: PlanoMinimo
   }

   export interface BibliotecaFiltros {
     q?: string
     tipo?: TipoConteudo
     regiao?: string
     de?: string
     ate?: string
   }

   export interface BibliotecaResponse {
     data: ConteudoCard[]
     next_cursor: number | null
   }
   ```

3. **`src/hooks/useBiblioteca.ts`**:
   - Usa `useInfiniteQuery` do TanStack Query
   - Chave: `['biblioteca', filtros]`
   - Query function: `GET /api/biblioteca` com filtros + cursor como query params
   - `getNextPageParam`: retorna `next_cursor` da última página (ou `undefined` se `null`)
   - Expõe: `conteudos` (array flat de `ConteudoCard`), `fetchNextPage`, `hasNextPage`, `isFetchingNextPage`, `isLoading`, `isError`

4. **`src/hooks/useConteudo.ts`**:
   - Usa `useQuery` do TanStack Query
   - Chave: `['conteudo', slug]`
   - Query function: `GET /api/biblioteca/:slug`
   - Trata erro 404: retorna `{ data: null, isPlanGate: true }` quando status 404
   - Expõe: `conteudo`, `isLoading`, `isError`, `isPlanGate`

### Regras de negócio aplicáveis:
- Erro 404 do endpoint individual indica acesso insuficiente ao plano → mostrar `PlanGate`
- Paginação usa cursor (id), não página numérica
- Stale time: 5 minutos para listagem, 10 minutos para leitura individual

### Critérios de aceitação:
- [ ] `useBiblioteca({ tipo: 'briefing' })` retorna lista de cards
- [ ] `useBiblioteca` com scroll infinito carrega próxima página via `fetchNextPage`
- [ ] `useConteudo('slug-inexistente')` expõe `isPlanGate: true`
- [ ] TypeScript compila sem erros de tipo nos hooks
- [ ] Dependências TipTap instaladas em `package.json`

# Task 8.4 – Páginas Biblioteca e ConteudoLeitura

## Objetivo
Criar as duas páginas públicas do assinante: a listagem com busca, filtros e scroll infinito, e a página de leitura individual do conteúdo.

## Tipo
frontend-pagina

## Arquivos a criar/modificar
- `src/pages/dashboard/Biblioteca.tsx` → criar
- `src/pages/dashboard/ConteudoLeitura.tsx` → criar
- `src/router.tsx` → modificar (adicionar rotas `/dashboard/biblioteca` e `/dashboard/biblioteca/:slug`)

## Contexto necessário (arquivos para ler antes)
- `src/hooks/useBiblioteca.ts` → assinatura do hook de listagem infinita
- `src/hooks/useConteudo.ts` → assinatura do hook de leitura individual
- `src/components/biblioteca/ContentCard.tsx` → interface de props
- `src/components/biblioteca/PlanGate.tsx` → interface de props
- `src/pages/dashboard/Feed.tsx` → ver padrão de página de listagem com scroll infinito

## Skills necessárias
- frontend-design-system
- animacoes-motion

## Especificação

### O que fazer:

1. **`Biblioteca.tsx`** (rota: `/dashboard/biblioteca`):
   - Estado local: `filtros: BibliotecaFiltros` (inicia vazio)
   - `SearchBar` no topo — ao mudar chama `setFiltros(f => ({ ...f, q: valor }))`
   - `FilterBar` abaixo da busca — ao mudar chama `setFiltros(novos)`
   - Grid de `ContentCard` com scroll infinito:
     - `IntersectionObserver` no último card (ou elemento sentinela) para chamar `fetchNextPage`
     - Skeleton loading para o primeiro carregamento (3 cards placeholder)
     - Spinner sutil ao carregar próxima página
   - Mensagem "Nenhum conteúdo encontrado" quando lista vazia e sem loading
   - Cada card navega para `/dashboard/biblioteca/:slug` ao clicar

2. **`ConteudoLeitura.tsx`** (rota: `/dashboard/biblioteca/:slug`):
   - Lê parâmetro `slug` via `useParams()`
   - Se `isPlanGate`: renderiza `<PlanGate tipo={conteudo?.tipo} />`
   - Se `isLoading`: skeleton de leitura (título + 5 linhas de texto)
   - Se `isError`: mensagem de erro com botão "Voltar"
   - Se conteúdo carregado: renderiza `<ContentReader corpo={conteudo.corpo} titulo={conteudo.titulo} />`
   - Breadcrumb no topo: "Biblioteca" (link) → título truncado
   - Botão "← Voltar" no topo esquerdo

3. **Rotas em `src/router.tsx`**:
   - `/dashboard/biblioteca` → `<Biblioteca />`
   - `/dashboard/biblioteca/:slug` → `<ConteudoLeitura />`
   - Ambas dentro do guard de autenticação existente

### Regras de negócio aplicáveis:
- Scroll infinito usa `IntersectionObserver` no sentinela final, não botão "carregar mais"
- `PlanGate` exibido quando 404 — nunca 403
- Filtros alteram o estado local; o hook reage automaticamente (TanStack Query)
- Mudar os filtros reseta a paginação (nova query key)

### Critérios de aceitação:
- [ ] `/dashboard/biblioteca` carrega e exibe lista de cards
- [ ] Busca com debounce filtra os resultados
- [ ] Scroll até o fim da página carrega mais itens automaticamente
- [ ] Clicar em card navega para `/dashboard/biblioteca/:slug`
- [ ] `/dashboard/biblioteca/:slug` exibe conteúdo completo ou `PlanGate`
- [ ] Rotas aparecem em `router.tsx`

## Teste esperado
- Verificação manual no browser: navegar em `/dashboard/biblioteca`, buscar, filtrar, rolar até o fim

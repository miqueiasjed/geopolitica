# Task 8.2 – ContentCard e PlanGate

## Objetivo
Criar o card de conteúdo com badge colorido por tipo e o componente de gate de plano exibido quando o backend retorna 404 por acesso insuficiente.

## Tipo
frontend-componente

## Arquivos a criar/modificar
- `src/components/biblioteca/ContentCard.tsx` → criar
- `src/components/biblioteca/PlanGate.tsx` → criar

## Contexto necessário (arquivos para ler antes)
- `src/types/biblioteca.ts` → tipos `ConteudoCard` e `TipoConteudo`
- `src/components/feed/EventCard.tsx` → ver padrão visual de card no projeto

## Skills necessárias
- frontend-design-system
- animacoes-motion

## Especificação

### O que fazer:

1. **`ContentCard.tsx`** (props: `conteudo: ConteudoCard`, `onClick?: () => void`):
   - Badge de tipo no canto superior esquerdo:
     - `briefing` → `bg-blue-500/20 text-blue-400 ring-1 ring-blue-500/40` + texto "Briefing"
     - `mapa` → `bg-yellow-500/20 text-yellow-400 ring-1 ring-yellow-500/40` + texto "Mapa"
     - `tese` → `bg-[#C9B882]/20 text-[#C9B882] ring-1 ring-[#C9B882]/40` + texto "A Tese"
   - Data `publicado_em` formatada (ex: "12 abr 2026")
   - Região (se existir) como tag pequena ao lado da data
   - Título em destaque
   - Se `tipo === 'tese'` e `tese_manchete` existir: exibir `tese_manchete` em itálico com `text-[#C9B882]` abaixo do título
   - Resumo em `text-sm text-zinc-400`, limitado a 3 linhas (`line-clamp-3`)
   - Tags como pequenas pills (se existirem)
   - Hover com `transition-colors` e borda sutil destacando
   - Animação de entrada com Framer Motion (`fadeUp` variant)

2. **`PlanGate.tsx`** (props: `tipo?: TipoConteudo`):
   - Exibido quando o conteúdo existe mas o plano é insuficiente
   - Ícone de cadeado centralizado
   - Título: "Conteúdo exclusivo" ou similar
   - Subtítulo explicando qual plano é necessário para o tipo do conteúdo
   - CTA button "Fazer upgrade" (link para página de planos)
   - Background: `bg-zinc-900/50 border border-zinc-800 rounded-xl`

### Regras de negócio aplicáveis:
- `tese_manchete` só aparece no card quando `tipo === 'tese'`
- Cor dourada `#C9B882` é exclusiva de "A Tese"
- `PlanGate` nunca revela detalhes do conteúdo bloqueado

### Critérios de aceitação:
- [ ] Badge de cor correta para cada tipo
- [ ] `tese_manchete` em itálico dourado visível apenas em cards do tipo `tese`
- [ ] `PlanGate` renderiza sem props (fallback genérico) e com `tipo` específico
- [ ] Animação de entrada funciona
- [ ] TypeScript compila sem erros

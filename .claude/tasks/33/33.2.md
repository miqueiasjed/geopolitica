# Task 33.2 – Tabelas Mobile: WebhookEventosPage, WebhookTokensPage, EleicaoPage (admin)

## Objetivo
Aplicar overflow-x-auto e wrapping de filtros nas tabelas de WebhookEventosPage, WebhookTokensPage e EleicaoPage (admin), eliminando overflow horizontal em 375px.

## Tipo
frontend-pagina

## Arquivos a criar/modificar
- `frontend/src/pages/admin/WebhookEventosPage.tsx` → modificar
- `frontend/src/pages/admin/WebhookTokensPage.tsx` → modificar
- `frontend/src/pages/admin/EleicaoPage.tsx` → modificar (versão admin)

## Contexto necessário (arquivos para ler antes)
- `frontend/src/pages/admin/WebhookEventosPage.tsx` → ver colunas (timestamp, tipo, payload, status) e filtros
- `frontend/src/pages/admin/WebhookTokensPage.tsx` → ver estrutura de colunas
- `frontend/src/pages/admin/EleicaoPage.tsx` → confirmar que é a versão admin (pode haver duplicidade com a versão assinante)

## Skills necessárias
- `frontend-design-system`

## Especificação

### O que fazer:
**Para cada uma das 3 páginas:**
1. Envolver `<table>` em `<div className="overflow-x-auto">` e adicionar `min-w-[560px]` na tabela.
2. WebhookEventosPage: coluna `payload` pode ter conteúdo longo — adicionar `max-w-[200px] truncate` na célula para não esticar a tabela além do mínimo.
3. Filtros horizontais: `flex flex-wrap gap-2`.
4. Botões de ação: texto com `hidden sm:inline`.
5. EleicaoPage (admin): verificar se há cards em grid — se sim, mudar para `grid-cols-1 sm:grid-cols-2` em mobile.

### Regras de negócio aplicáveis:
- Colunas com conteúdo potencialmente longo (payload, URLs) devem ter `truncate` para não esticar a tabela.
- `min-w-[560px]` é o mínimo para tabelas com 4+ colunas.

### Critérios de aceitação:
- [ ] WebhookEventosPage: tabela scrollável em 375px, payload truncado na célula.
- [ ] WebhookTokensPage: tabela scrollável em 375px.
- [ ] EleicaoPage (admin): tabela/grid scrollável ou responsivo em 375px.
- [ ] Filtros não transbordam em nenhuma das 3 páginas em mobile.

## Teste esperado
- Verificação visual em 375px: scroll lateral nas tabelas, sem bloqueio de conteúdo.

# Task 28.5 – Reescrever AddonGate.tsx com CTA dinâmico por status

## Objetivo
Reescrever o componente `AddonGate.tsx` para exibir CTA dinâmico baseado no `status_usuario` do produto, usando dados do hook `useProdutos` e preservando o design atual.

## Tipo
frontend-componente

## Arquivos a criar/modificar
- `frontend/src/components/AddonGate.tsx` → modificar

## Contexto necessário (arquivos para ler antes)
- `frontend/src/components/AddonGate.tsx` → entender design e lógica atual
- `frontend/src/hooks/useProdutos.ts` → usar `useProdutos`, `resolveLinkCta`, `resolveLabelCta`
- `frontend/src/types/produto.ts` → tipos

## Skills necessárias
- `frontend-design-system`
- `animacoes-motion`

## Especificação

### O que fazer:
**Props:** `{ chave: string; children: React.ReactNode }`

**Lógica de CTA por status:**
| status_usuario | Link usado | Label do botão |
|---|---|---|
| `null` | `link_compra` | "Adicionar" |
| `'cancelado'` | `link_reativar` ?? `link_compra` | "Reativar" |
| `'expirado'` | `link_compra` ?? `link_reativar` | "Renovar" |
| `'reembolsado'` | `link_compra` | "Assinar" |
| Ambos links null | (desabilitado) | "Entre em contato" |
| `'ativo'` | — | (renderizar children) |

**Comportamento:**
- Se `status_usuario === 'ativo'`: renderizar `children` diretamente
- Caso contrário: exibir card de bloqueio com CTA correspondente
- Card de bloqueio: design atual preservado (fundo escuro, `preco_label`, botão verde-limão `#B5E550` ou similar)
- Botão desabilitado quando ambos links são null: texto "Entre em contato", opacidade reduzida

**Estado de loading:** enquanto `useProdutos` carrega, não renderizar children (evitar flash de conteúdo)

### Regras de negócio aplicáveis:
- Usar `resolveLinkCta` e `resolveLabelCta` do `useProdutos`
- Design atual preservado — não criar nova identidade visual
- Botão desabilitado não deve ter `href`/`onClick`

### Critérios de aceitação:
- [ ] Addon `ativo` → children renderizados
- [ ] Addon `null` → card com botão "Adicionar" + link_compra
- [ ] Addon `cancelado` → card com botão "Reativar" + link_reativar
- [ ] Ambos links null → botão "Entre em contato" desabilitado
- [ ] Design do card idêntico ao atual
- [ ] Sem flash de conteúdo durante loading

## Teste esperado
- Verificação visual: navegar para Monitor Eleitoral/Guerra com diferentes estados de addon
- Confirmar cada CTA abre link correto

# Task 10.7 – AlertaPanel e integração na TopNav

## Objetivo
Criar o painel deslizante de alertas com lista, badges de nível e botão "Marcar como lido", e integrar o `AlertaBadge` na `TopNav`.

## Tipo
frontend-componente

## Arquivos a criar/modificar
- `src/components/alertas/AlertaPanel.tsx` → criar
- `src/components/layout/TopNav.tsx` → modificar (adicionar AlertaBadge)

## Contexto necessário (arquivos para ler antes)
- `src/types/alertas.ts` → tipos `Alerta` e `NivelAlerta`
- `src/hooks/useAlertas.ts` → assinatura do hook
- `src/hooks/useMarcarAlertaLido.ts` → assinatura da mutation
- `src/components/alertas/AlertaBadge.tsx` → interface e como controla abertura do painel
- `src/components/layout/TopNav.tsx` → estrutura atual para inserir o badge

## Skills necessárias
- frontend-design-system
- animacoes-motion

## Especificação

### O que fazer:

1. **`AlertaPanel.tsx`** (props: `isOpen: boolean`, `onClose: () => void`):
   - Painel lateral deslizante a partir da direita (Framer Motion `x: '100%'` → `x: 0`)
   - `position: fixed`, `top-0 right-0 h-full w-80 md:w-96`
   - Background: `bg-zinc-950 border-l border-zinc-800`
   - Header: "Alertas" com botão fechar (×)
   - Overlay semi-transparente ao fundo quando aberto
   - Lista de alertas (de `useAlertas`):
     - Badge de nível por item:
       - `critical` → `bg-red-500/20 text-red-400 ring-1 ring-red-500/40` + texto "Crítico"
       - `high` → `bg-orange-500/20 text-orange-400 ring-1 ring-orange-500/40` + texto "Alto"
       - `medium` → `bg-yellow-500/20 text-yellow-400 ring-1 ring-yellow-500/40` + texto "Médio"
     - Região + data
     - Título do alerta
     - Primeiros 100 chars da análise (truncado com "...")
     - Botão "Marcar como lido" — chama `useMarcarAlertaLido`:
       - Ao clicar, o item desaparece da lista (o hook invalida a query)
       - Animação de saída: `AnimatePresence` + `exit: { opacity: 0, x: 20 }`
   - Estado vazio: "Nenhum alerta pendente 🎉"
   - Skeleton loading enquanto carrega

2. **`TopNav.tsx`** — adicionar `AlertaBadge`:
   - Importar e inserir `<AlertaBadge onTogglePanel={handleTogglePanel} />` na área direita da nav
   - Gerenciar estado `isPanelOpen` localmente na TopNav
   - Renderizar `<AlertaPanel isOpen={isPanelOpen} onClose={() => setIsPanelOpen(false)} />` como filho direto do `body` (via `portal`) ou ao final do componente

### Regras de negócio aplicáveis:
- `AnimatePresence` necessário para animação de saída dos itens marcados como lidos
- Painel não bloqueia scroll do conteúdo de fundo
- Badge some quando `totalNaoLidos === 0` (não precisa de interação para fechar)

### Critérios de aceitação:
- [ ] Clicar no sino abre o painel com animação deslizante
- [ ] Lista exibe alertas com badge de nível correto
- [ ] "Marcar como lido" remove o item com animação de saída
- [ ] Badge na TopNav atualiza contagem após marcar como lido
- [ ] Fechar painel via overlay ou botão × funciona
- [ ] Estado vazio exibido quando não há alertas

## Teste esperado
- Verificação manual no browser: abrir painel, marcar alertas, verificar que badge atualiza

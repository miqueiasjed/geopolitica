# Task 24.6 – Componente ScoreGauge: gauge de risco por categoria

## Objetivo
Criar o componente reutilizável `ScoreGauge.tsx` que exibe um score de risco de 0-100 com barra de progresso animada, valor numérico, label da categoria e badge BAIXO/MÉDIO/ALTO com cor dinâmica.

## Tipo
frontend-componente

- `frontend/src/components/ScoreGauge.tsx` → criar

- `frontend/src/components/AddonGate.tsx` → ver padrão de estilo de cards no projeto

## Especificação

### O que fazer:
```typescript
interface ScoreGaugeProps {
  valor: number      // 0-100
  label: string      // ex: "Energia", "Alimentos"
  className?: string
}
```

Lógica de cor por nível:
- `valor >= 70` → ALTO → `#EF4444` (vermelho)
- `valor >= 40` → MÉDIO → `#FACC15` (amarelo)
- `valor < 40`  → BAIXO → `#4ade80` (verde)

Layout do card (fundo `#0d0d0f`, borda `border-white/8`):
```
[Label categoria]                    [ALTO/MÉDIO/BAIXO]
[════════════════════░░░░░░░]         ← barra progresso
[72]                /100             ← valor em mono
```

Barra de progresso:
- Container: `h-1.5 bg-white/5 rounded-full overflow-hidden`
- Fill: `h-full rounded-full transition-all duration-1000` com `width: {valor}%` e `backgroundColor: corNivel`

Animação de entrada com Framer Motion:
```tsx
<motion.div
  initial={{ opacity: 0, y: 8 }}
  animate={{ opacity: 1, y: 0 }}
  transition={{ duration: 0.3 }}
>
```

Badge de nível:
- `text-[10px] tracking-wider uppercase font-medium` com `color: corNivel`

Valor numérico:
- `text-2xl font-mono font-medium` com `color: corNivel`
- `/100` em `text-sm text-white/30`

### Regras de negócio aplicáveis:
- Valor `0` exibe barra vazia e badge BAIXO em verde.
- Valor `100` exibe barra cheia e badge ALTO em vermelho.
- Animação da barra via CSS transition (não Framer Motion) para ser mais performática em 4 gauges simultâneos.

### Critérios de aceitação:
- [ ] Card com label, badge de nível, barra de progresso e valor numérico
- [ ] Cor muda corretamente conforme limites (< 40 verde, 40-69 amarelo, >= 70 vermelho)
- [ ] Barra anima suavemente quando o valor muda
- [ ] Animação de entrada com Framer Motion
- [ ] Funciona com valores 0, 40, 70 e 100 sem quebrar layout

# Task 24.4 – RiskScoreService: motor de cálculo e mapeamento por IA

## Especificação

### O que fazer:
Criar `App\Services\RiskScoreService` com os seguintes métodos:

#### `calcularTensaoPorCategoria(): array`
Busca eventos ativos das últimas 48h:
```php
$eventos = DB::table('eventos')
    ->where('is_active', true)
    ->where('published_at', '>=', now()->subHours(48))
    ->select('impact_score', 'categories')
    ->get();
```
Mapeamento de categorias do feed para categorias de risco:
```php
private const MAPA_CATEGORIAS = [
    'energy'     => 'energy', 'oil' => 'energy', 'gas' => 'energy',
    'food'       => 'food',   'agriculture' => 'food', 'grain' => 'food',
    'currency'   => 'fx',     'trade' => 'fx', 'sanctions' => 'fx',
    'military'   => 'military', 'war' => 'military', 'conflict' => 'military',
];
```
Para cada evento: `$categorias = json_decode($evento->categories, true) ?? []`. Para cada categoria, se mapeada, acumular `impact_score` e contar. Ao final, normalizar: `min(100, ($soma / $contagem) * 10)`. Valor padrão quando sem eventos: energy=30, food=25, fx=35, military=40.

#### `calcularRiscoPortfolio(array $ativos): array`
`$ativos` = `[['ticker' => 'PETR4', 'peso' => 0.3], ...]` (pesos já em decimal 0-1).
1. `$tensao = $this->calcularTensaoPorCategoria()`
2. Buscar mapeamentos: `MapaRiscoAtivo::whereIn('ticker', $tickers)->get()->keyBy('ticker')`
3. Para ativos não encontrados: `$this->mapearAtivoComIA($naoMapeados)`
4. Para cada ativo: `$score[$cat] += $tensao[$cat] * $riskWeights[$cat] * $ativo['peso']`
5. Normalizar dividindo pela soma dos pesos
6. `$total = round(($energia + $alimentos + $cambio + $militar) / 4)`
7. Buscar 3 alertas preditivos ativos: `AlertaPreditivo::where('is_active', true)->orderByDesc('created_at')->limit(3)->get()`
8. Retornar:
```php
return [
    'total'       => $total,             // 0-100
    'breakdown'   => [                   // 0-100 por categoria
        'energia'   => round($scores['energy']),
        'alimentos' => round($scores['food']),
        'cambio'    => round($scores['fx']),
        'militar'   => round($scores['military']),
    ],
    'alertas'      => $alertas,          // lista de alertas relacionados
    'top_riscos'   => $topRiscos,        // 2 categorias com maior score
    'calculado_em' => now()->toISOString(),
];
```

#### `mapearAtivoComIA(array $tickers): Collection`
Se `$tickers` estiver vazio, retornar `collect([])`.
Montar prompt:
```
Para cada ativo abaixo, forneça os pesos de risco geopolítico (0.0 a 1.0).
Categorias: energy (petróleo, gás), food (agro, grãos), fx (câmbio, comércio, sanções), military (defesa, conflitos)
Ativos: {tickers}
Retorne APENAS JSON: [{"ticker":"X","name":"Nome","asset_type":"stock_br","risk_weights":{"energy":0.0,"food":0.0,"fx":0.0,"military":0.0}}]
```
Chamar `AiProviderFactory::make()->completar($prompt, maxTokens: 600)`. Parsear JSON da resposta. Salvar cada item via `MapaRiscoAtivo::upsert()` com `updatedAt = now()`. Retornar como Collection indexada por ticker.

#### `salvarCarteira(int $userId, array $ativos): Carteira`
- Validar pesos: `array_sum(array_column($ativos, 'peso'))` deve estar entre 0.95 e 1.05
- Calcular score: `$score = $this->calcularRiscoPortfolio($ativos)`
- `Carteira::updateOrCreate(['user_id' => $userId], ['ativos' => $ativos, 'ultimo_score' => $score, 'updated_at' => now()])`
- Retornar `Carteira` com score calculado e adicionado como propriedade temporária `->score = $score`

### Regras de negócio aplicáveis:
- Pesos de portfólio chegam em decimal (0.3 = 30%). Validar soma entre 0.95–1.05.
- Normalização do score: nunca pode ultrapassar 100 nem ser negativo.
- `mapearAtivoComIA` só é chamada se houver tickers não encontrados no banco.
- Salvar mapeamentos da IA no banco para cache — evitar chamadas repetidas.

### Critérios de aceitação:
- [ ] `calcularTensaoPorCategoria()` retorna valores 0-100 para as 4 categorias
- [ ] `calcularRiscoPortfolio()` retorna breakdown e total corretos para portfólio simples
- [ ] `mapearAtivoComIA()` é chamada apenas para tickers não no banco
- [ ] Ativos mapeados pela IA são salvos no banco (cache)
- [ ] `salvarCarteira()` valida pesos e persiste carteira com score
- [ ] Score total sempre entre 0 e 100

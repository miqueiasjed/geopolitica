# Task 22.7 – Integrar ExportPdfButton nas páginas existentes

## Objetivo
Adicionar o `ExportPdfButton` nas três páginas que suportam exportação: leitura de briefing, perfil de país e chat, passando o `tipo` e `id` corretos para cada contexto.

## Tipo
frontend-pagina

## Arquivos a criar/modificar
- `frontend/src/pages/dashboard/ConteudoLeitura.tsx` → modificar
- `frontend/src/pages/PerfilPaisPage.tsx` → modificar
- `frontend/src/pages/ChatBriefings.tsx` → modificar

## Contexto necessário (arquivos para ler antes)
- `frontend/src/pages/dashboard/ConteudoLeitura.tsx` → ver onde o conteúdo é exibido e qual campo carrega o ID
- `frontend/src/pages/PerfilPaisPage.tsx` → ver onde o perfil é exibido e qual campo carrega o ID/código
- `frontend/src/pages/ChatBriefings.tsx` → ver estrutura das mensagens e onde adicionar o botão por resposta

## Skills necessárias
- frontend-design-system

## Especificação

### O que fazer:

#### `ConteudoLeitura.tsx`
- Importar `ExportPdfButton`
- Localizar o cabeçalho ou ações da página (próximo ao título do conteúdo)
- Adicionar `<ExportPdfButton tipo="briefing" id={conteudo.id} label="Exportar PDF" />`
- Posicionar no canto superior direito do cabeçalho ou abaixo do título, alinhado com outras ações existentes

#### `PerfilPaisPage.tsx`
- Importar `ExportPdfButton`
- Localizar o header do perfil de país
- Adicionar `<ExportPdfButton tipo="pais" id={pais.id ?? pais.codigo} label="Exportar perfil" />`

#### `ChatBriefings.tsx`
- Importar `ExportPdfButton`
- Localizar onde cada mensagem de resposta da IA é renderizada (componente `ChatMessage` ou similar)
- Para cada mensagem com `role === 'assistant'`, adicionar botão de exportação com `tipo="chat"` e `id={mensagem.id}`
- O botão deve ser discreto (pequeno, abaixo da resposta) sem interferir no fluxo da conversa

### Regras de negócio aplicáveis:
- No chat, exibir o botão apenas em mensagens com `role === 'assistant'` (respostas da IA).
- Não bloquear a UI durante geração — o botão tem estado de loading próprio.
- Manter o layout existente das páginas; o botão deve ser adicionado sem quebrar o design.

### Critérios de aceitação:
- [ ] `ExportPdfButton` visível em `ConteudoLeitura` com `tipo="briefing"` e ID correto
- [ ] `ExportPdfButton` visível em `PerfilPaisPage` com `tipo="pais"` e ID correto
- [ ] `ExportPdfButton` visível em cada resposta do assistente em `ChatBriefings` com `tipo="chat"`
- [ ] Nenhuma quebra de layout nas três páginas
- [ ] Download funciona corretamente ao clicar em cada página

## Teste esperado
- Verificação manual: abrir cada página no browser, clicar no botão e verificar que o PDF é baixado
- Confirmar que o PDF gerado contém o conteúdo correto para cada tipo

## Estimativa de complexidade
baixa
~20 linhas de modificação | ~3 arquivos

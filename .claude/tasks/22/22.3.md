# Task 22.3 – Blade templates PDF: perfil de país e chat

## Objetivo
Criar os dois templates Blade restantes para geração de PDFs: perfil de país e resposta do chat.

## Tipo
backend-estrutura

## Arquivos a criar/modificar
- `backend/resources/views/pdf/pais.blade.php` → criar
- `backend/resources/views/pdf/chat.blade.php` → criar

## Contexto necessário (arquivos para ler antes)
- `backend/app/Models/PerfilPais.php` → entender campos disponíveis
- `backend/app/Models/ChatMensagem.php` → entender estrutura da mensagem (pergunta + resposta)
- `backend/resources/views/pdf/layout.blade.php` → ver como fazer o extend/include do layout

## Skills necessárias
- laravel-arquitetura

## Especificação

### O que fazer:

#### `pdf/pais.blade.php`
Recebe variável `$pais` (Model PerfilPais). Exibe as seções do perfil:
- Nome do país + código ISO como subtítulo
- Seção "Contexto Geopolítico": campo `contexto_geral`
- Seção "Liderança Atual": campos de liderança
- Seção "Principais Tensões": campo `tensoes_atuais`
- Seção "Indicadores de Risco": campos numéricos (se disponíveis) em tabela simples
- Seção "O Que Monitorar": campo `pontos_monitoramento`
- Cada seção separada por `border-bottom: 0.5px solid #eeeeee`

#### `pdf/chat.blade.php`
Recebe variável `$mensagem` (Model ChatMensagem com eager load da sessão). Exibe:
- Label "Pergunta" + texto da pergunta em destaque (border-left: 2px solid #C9B882)
- Label "Resposta da IA" + corpo da resposta em Markdown convertido para HTML (usar `{!! nl2br(e($mensagem->resposta)) !!}` ou similar)
- Seção "Fontes Consultadas" se `$mensagem->fontes` não for nulo: lista de fontes com título e tipo
- Data e hora da conversa

### Regras de negócio aplicáveis:
- Todo CSS inline/`<style>` — sem dependências externas.
- Para a resposta do chat, converter quebras de linha em `<br>` — DomPDF não processa Markdown nativo.
- Fontes do chat são armazenadas como JSON; iterar com `@foreach($mensagem->fontes ?? [] as $fonte)`.

### Critérios de aceitação:
- [ ] `pdf/pais.blade.php` exibe todas as seções do perfil com separadores visuais
- [ ] `pdf/chat.blade.php` exibe pergunta em destaque, resposta formatada e fontes
- [ ] Ambos estendem/incluem o layout base (`pdf/layout.blade.php`)
- [ ] Nenhum CSS externo referenciado

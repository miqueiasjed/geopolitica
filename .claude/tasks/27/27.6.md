# Task 27.6 – Endpoint Exportar Addons (GET /admin/assinantes/addons/exportar)

## Objetivo
Implementar o endpoint de exportação CSV com todos os addons ativos, com as colunas: email, nome, addon_key, status, fonte, iniciado_em, expira_em.

## Tipo
backend-endpoint

## Arquivos a criar/modificar
- `backend/app/Http/Controllers/Api/Admin/AdminAssinanteAddonController.php` → modificar

## Contexto necessário (arquivos para ler antes)
- `backend/app/Http/Controllers/Api/Admin/AdminAssinanteAddonController.php` → estado após Task 27.5
- `backend/app/Models/AssinanteAddon.php` → ver relacionamentos (user, etc)

## Skills necessárias
- `laravel-arquitetura`
- `datas-timezone`

## Especificação

### O que fazer:
Adicionar método `exportar()` ao controller:

```php
public function exportar(): StreamedResponse
{
    $addons = AssinanteAddon::with('user')
        ->where('status', 'ativo')
        ->orderBy('criado_em', 'desc')
        ->get();

    $cabecalhos = ['email', 'nome', 'addon_key', 'status', 'fonte', 'iniciado_em', 'expira_em'];

    $nomeArquivo = 'addons_ativos_' . now('America/Sao_Paulo')->format('Y-m-d_His') . '.csv';

    return response()->streamDownload(function () use ($addons, $cabecalhos) {
        $fp = fopen('php://output', 'w');
        fputcsv($fp, $cabecalhos);

        foreach ($addons as $addon) {
            fputcsv($fp, [
                $addon->user->email,
                $addon->user->name,
                $addon->addon_key,
                $addon->status,
                $addon->fonte,
                $addon->iniciado_em?->format('Y-m-d'),
                $addon->expira_em?->format('Y-m-d'),
            ]);
        }

        fclose($fp);
    }, $nomeArquivo, ['Content-Type' => 'text/csv']);
}
```

**Rota:** `GET /admin/assinantes/addons/exportar` no grupo admin.

### Regras de negócio aplicáveis:
- Exportar apenas addons com `status=ativo`
- Datas formatadas em `Y-m-d` (sem timezone no CSV)
- Nome do arquivo inclui data/hora em `America/Sao_Paulo`

### Critérios de aceitação:
- [ ] `GET /admin/assinantes/addons/exportar` retorna download de arquivo CSV
- [ ] CSV contém as 7 colunas corretas no cabeçalho
- [ ] Apenas addons ativos são incluídos
- [ ] Nome do arquivo contém data no formato `Y-m-d_His`
- [ ] Usuário não-admin recebe 403

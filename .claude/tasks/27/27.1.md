# Task 27.1 – Instalar maatwebsite/excel + Criar ImportarAddonsJob

## Objetivo
Instalar a dependência `maatwebsite/excel` via Composer e criar o Job `ImportarAddonsJob` para processamento assíncrono de importações com mais de 500 linhas.

## Tipo
backend-estrutura

## Arquivos a criar/modificar
- `backend/composer.json` → modificar (adicionar maatwebsite/excel)
- `backend/app/Jobs/ImportarAddonsJob.php` → criar

## Contexto necessário (arquivos para ler antes)
- `backend/composer.json` → ver dependências atuais e versão do Laravel
- `backend/app/Jobs/` → ver se existe padrão de Job no projeto

## Skills necessárias
- `laravel-arquitetura`

## Especificação

### O que fazer:
1. Instalar: `composer require maatwebsite/excel`
2. Publicar config se necessário: `php artisan vendor:publish --provider="Maatwebsite\Excel\ExcelServiceProvider" --tag=config`

**ImportarAddonsJob** (`implements ShouldQueue`):

```php
class ImportarAddonsJob implements ShouldQueue

    public function __construct(
        private string $caminhoArquivo,
        private string $mimeType,
        private int    $adminUserId,
    ) {}

    public function handle(AddonService $addonService): void
    {
        // processar linhas via maatwebsite/excel
        // chamar lógica de importação compartilhada com o endpoint síncrono
        // armazenar resultado em cache (ex: cache()->put("import_result_{$this->job->getJobId()}", ...))
    }

```

O job deve:
- Receber o caminho do arquivo temporário, mime type e ID do admin
- Processar linhas usando lógica idêntica ao endpoint síncrono
- Armazenar resultado em cache para consulta posterior (opcional nesta task — pode ser stub)

### Regras de negócio aplicáveis:
- Arquivos > 500 linhas devem ser enfileirados, não processados no request
- O job reutiliza a mesma lógica de importação do endpoint síncrono

### Critérios de aceitação:
- [ ] `composer require maatwebsite/excel` instalado sem erros
- [ ] Classe `ImportarAddonsJob` existe e implementa `ShouldQueue`
- [ ] Job pode ser despachado sem erros: `ImportarAddonsJob::dispatch(...)`
- [ ] `php artisan queue:work` não apresenta erros ao processar o job (mesmo que stub)

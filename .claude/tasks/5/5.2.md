# Task 5.2 – Componentes ImpactBadge e EventCard

## Objetivo
Criar o componente `ImpactBadge` com cores por nível de impacto e o `EventCard` completo com score, badge, título, análise IA e metadados no estilo Bloomberg Terminal.

## Tipo
frontend-componente

## Arquivos a criar/modificar
- `frontend/src/components/ImpactBadge.tsx` → criar
- `frontend/src/components/EventCard.tsx` → criar

## Contexto necessário (arquivos para ler antes)
- `frontend/src/types/feed.ts` → ver interfaces Event e ImpactLabel (task 5.1)

## Skills necessárias
- frontend-design-system
- animacoes-motion

## Especificação

### O que fazer:
1. Criar `ImpactBadge.tsx`:
   - Props: `label: ImpactLabel`, `score?: number`
   - Mapeamento de cores:
     - `CRÍTICO` → `bg-red-600/20 text-red-400 border border-red-600/40` (`#EF4444`)
     - `ALTO` → `bg-orange-600/20 text-orange-400 border border-orange-600/40` (`#FB923C`)
     - `MÉDIO` → `bg-yellow-600/20 text-yellow-400 border border-yellow-600/40` (`#FACC15`)
     - `MONITORAR` → `bg-blue-600/20 text-blue-400 border border-blue-600/40` (`#60A5FA`)
   - Exibir label + score entre parênteses se fornecido: `ALTO (7)`
   - Tamanho: `text-xs px-2 py-0.5 rounded font-mono uppercase`
2. Criar `EventCard.tsx`:
   - Props: `event: Event`
   - Layout:
     - Header: score numérico grande (monospace, cor do impacto) + `ImpactBadge` + fonte + data relativa
     - Título: link (se `fonte_url` disponível), `text-white font-medium`
     - Análise IA: texto `text-gray-400 text-sm` (mostrar se `analise_ia` não null)
     - Footer: regiões como chips + categorias como chips dourados `#C9B882`
   - Fundo: `bg-[#111113] border border-[#1e1e20]`
   - Hover: `hover:border-[#C9B882]/30 transition-colors`
   - Animação de entrada: `framer-motion` com `initial={{ opacity: 0, y: 8 }}` e `animate={{ opacity: 1, y: 0 }}`

### Regras de negócio aplicáveis:
- Cor dourada `#C9B882` deve aparecer apenas nos chips de categoria e hover
- O score deve ser exibido com a cor correspondente ao impact_label
- Tipografia monospace para dados numéricos (score, data)

### Critérios de aceitação:
- [ ] `ImpactBadge` exibe badge vermelho para CRÍTICO, laranja para ALTO, etc.
- [ ] `EventCard` exibe score, badge, título, análise e metadados
- [ ] Chips de categoria usam a cor dourada `#C9B882`
- [ ] Hover anima border para dourado
- [ ] Animação de entrada funciona sem layout shift

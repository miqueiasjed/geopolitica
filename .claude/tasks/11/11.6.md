# Task 11.6 – BuscaPais, CardPais e MeusPaisesPage

## Objetivo
Implementar o componente de busca com debounce, o card de país seguido e a página "Meus Países" com lista + busca para adicionar novos países.

## Tipo
frontend-pagina

## Arquivos a criar/modificar
- `src/components/paises/BuscaPais.tsx` → criar
- `src/components/paises/CardPais.tsx` → criar
- `src/pages/MeusPaisesPage.tsx` → criar
- `src/router.tsx` (ou equivalente) → modificar (adicionar rota `/paises`)

## Contexto necessário (arquivos para ler antes)
- `src/hooks/useMeusPaises.ts` → mutations adicionarPais/removerPais (11.5)
- `src/types/pais.ts` → PaisUsuario, PaisResumo (11.5)
- `src/components/` → ver padrão de componentes existentes (nomenclatura, estrutura)
- `src/pages/` → ver estrutura de páginas do projeto
- `src/router.tsx` → ver como rotas são declaradas

## Skills necessárias
- frontend-design-system

## Especificação

### O que fazer:

**`BuscaPais.tsx`:**
Props: `onSelect: (codigoPais: string) => void`
- Input controlado com `useState`
- Debounce de 300ms no termo de busca (usar `useDebounce` do projeto ou implementar com `useEffect` + `setTimeout`)
- Quando `q.length >= 2`: fazer `GET /api/paises?q={termo}` via `useQuery` interno
- Exibir dropdown com resultados: bandeira + nome + região
- Ao clicar no resultado: chamar `onSelect(pais.codigo_pais)` e limpar o input
- Estado `isLoading` com skeleton no dropdown
- Mensagem "Nenhum país encontrado" se resultado vazio
- Estilo: input com borda dourada `border-[#C9B882]` no foco

**`CardPais.tsx`:**
Props: `paisUsuario: PaisUsuario`, `onRemover: (codigoPais: string) => void`
- Exibir: bandeira_emoji + nome_pt + regiao_geopolitica
- Badge com `adicionado_em` formatado (dd/MM/yyyy)
- Botão "Acompanhar" (link → `/paises/{codigo_pais}`)
- Botão "Remover" (ícone de lixeira) com `onClick: () => onRemover(pais.codigo_pais)`
- Botão Remover deve ter estado de loading durante a mutation
- Estilo: card com fundo escuro, borda sutil, hover com borda dourada

**`MeusPaisesPage.tsx`:**
- Título da página: "Meus Países"
- Seção superior: `BuscaPais` + botão "Adicionar" (disabled enquanto loading)
- Ao selecionar país na busca: chamar `adicionarPais`, exibir toast de sucesso ou erro de limite (422)
- Seção inferior: grid de `CardPais` com os países seguidos
- Empty state se lista vazia: "Você ainda não segue nenhum país. Use a busca acima para começar."
- Skeleton loading ao carregar a lista
- Exibir badge com contador: "X / 3 países" (essencial) ou "X / 10 países" (pro)

**Rota no router:**
```tsx
{ path: '/paises', element: <MeusPaisesPage /> }
```
(Dentro do layout protegido do dashboard)

### Regras de negócio aplicáveis:
- Exibir feedback de erro de limite ao tentar adicionar (mensagem do 422)
- Debounce de 300ms no campo de busca
- Busca só dispara com 2+ caracteres

### Critérios de aceitação:
- [ ] Busca com debounce 300ms funcionando
- [ ] Dropdown fecha ao selecionar país
- [ ] Card exibe corretamente bandeira + nome + região + data
- [ ] Remover país atualiza lista imediatamente
- [ ] Empty state aparece quando lista vazia
- [ ] Badge de contador de países exibido corretamente
- [ ] Toast de erro ao atingir limite do plano

## Teste esperado
- Verificação visual no browser: `/paises`
- Testar fluxo: buscar país → adicionar → verificar badge → remover → verificar lista

# Task 11.7 – PerfilPaisPage, integração no WorldMap e navegação

## Objetivo
Implementar a página de perfil individual do país com 5 seções, integrar o clique no WorldMap do M02 para redirecionar a países cadastrados, e adicionar "Meus Países" na navegação do dashboard.

## Tipo
frontend-pagina

## Arquivos a criar/modificar
- `src/pages/PerfilPaisPage.tsx` → criar
- `src/components/mapa/WorldMap.tsx` → modificar (integração clique → perfil)
- `src/components/layout/TopNav.tsx` → modificar (link "Meus Países")
- `src/router.tsx` (ou equivalente) → modificar (rota `/paises/:codigo`)

## Contexto necessário (arquivos para ler antes)
- `src/hooks/usePerfilPais.ts` → dados do perfil (11.5)
- `src/hooks/useEventosPais.ts` → eventos recentes (11.5)
- `src/hooks/useMeusPaises.ts` → para toggle acompanhar (11.5)
- `src/types/pais.ts` → PerfilPais, EventoPais (11.5)
- `src/components/mapa/WorldMap.tsx` → ver como o clique atual funciona (M02)
- `src/components/layout/TopNav.tsx` → ver estrutura da navegação

## Skills necessárias
- frontend-design-system

## Especificação

### O que fazer:

**`PerfilPaisPage.tsx`:**
- Ler `codigo` via `useParams()`
- Usar `usePerfilPais(codigo)` e `useEventosPais(codigo)`
- **Seção 1 — Header:** bandeira_emoji grande + nome_pt + regiao_geopolitica + botão toggle "Acompanhar / Parar de acompanhar"
  - Toggle usa `useMeusPaises().adicionarPais` / `removerPais`
  - Botão fica disabled enquanto mutation está pendente
- **Seção 2 — Contexto Geopolítico:** texto de `contexto_geopolitico`
  - Se null: mostrar skeleton animado + texto "Análise sendo gerada..."
- **Seção 3 — Liderança Atual:** texto de `analise_lideranca`
  - Se null: mostrar skeleton
- **Seção 4 — Indicadores Econômicos:** grid de badges com os `indicadores_relevantes`
  - Cada badge: nome do indicador com ícone de gráfico
  - Se lista vazia: ocultar seção
- **Seção 5 — Eventos Recentes:** lista de `EventoPais`
  - Cada item: `titulo` + `nivel_tensao` (badge colorido) + `created_at` relativo ("há 2 dias")
  - Se vazio: "Nenhum evento recente encontrado"
  - Link para o evento no feed (se aplicável)
- Skeleton loading geral enquanto perfil carrega
- Botão "← Voltar" para `/paises`

**Integração no `WorldMap.tsx`:**
- Importar `useNavigate` do React Router
- Importar lista de países do M11 (ou fazer query com `useQuery(['paises'])` para saber quais têm perfil)
- No handler de clique no país do mapa:
  ```typescript
  // Se o país clicado tem perfil cadastrado (existe em perfis_paises)
  if (paisTemPerfil(codigoPais)) {
    navigate(`/paises/${codigoPais}`);
  } else {
    // comportamento atual: abrir RegionPanel
    abrirRegionPanel(pais);
  }
  ```
- Carregar lista de `codigo_pais` disponíveis via `useQuery(['paises'])` no início do componente

**`TopNav.tsx`:**
- Adicionar link "Meus Países" na navegação principal do dashboard
- Posicionar próximo aos outros links de módulos
- Usar ícone de bandeira (ou globo) antes do texto

**Rota:**
```tsx
{ path: '/paises/:codigo', element: <PerfilPaisPage /> }
```

### Regras de negócio aplicáveis:
- Clique no mapa só redireciona se o país estiver nos 20 cadastrados
- Botão acompanhar reflete estado real (verificar se já está em `meus-paises`)
- Seções de conteúdo IA mostram skeleton se campo for null

### Critérios de aceitação:
- [ ] Página `/paises/BR` carrega com as 5 seções
- [ ] Skeleton exibido quando `contexto_geopolitico` é null
- [ ] Toggle "Acompanhar" funciona (adiciona/remove de meus-paises)
- [ ] Clique no Brasil no WorldMap → redireciona para `/paises/BR`
- [ ] Clique em país sem perfil → comportamento anterior (RegionPanel)
- [ ] Link "Meus Países" aparece no TopNav

- Testar toggle acompanhar

## Especificação

### O que fazer:

**`src/types/pais.ts`:**
```typescript
export interface PerfilPais {
  codigo_pais: string;
  nome_pt: string;
  bandeira_emoji: string | null;
  regiao_geopolitica: string;
  contexto_geopolitico: string | null;
  analise_lideranca: string | null;
  indicadores_relevantes: string[];
  termos_busca: string[];
  gerado_em: string | null;
}

export interface PaisResumo {
  codigo_pais: string;
  nome_pt: string;
  bandeira_emoji: string | null;
  regiao_geopolitica: string;
  gerado_em: string | null;
}

export interface PaisUsuario {
  id: number;
  user_id: number;
  codigo_pais: string;
  adicionado_em: string;
  perfil: PaisResumo;
}

export interface EventoPais {
  id: number;
  titulo: string;
  descricao: string;
  nivel_tensao: string;
  created_at: string;
}
```

**`useMeusPaises`:**
- Query key: `['meus-paises']`
- Fetch: `GET /api/meus-paises`
- Retornar: `{ data: PaisUsuario[], isLoading, error }`
- Incluir mutations:
  - `adicionarPais(codigo_pais: string)` → `POST /api/meus-paises` + invalidar `['meus-paises']`
  - `removerPais(codigo_pais: string)` → `DELETE /api/meus-paises/{codigo}` + invalidar `['meus-paises']`

**`usePerfilPais`:**
- Parâmetro: `codigoPais: string`
- Query key: `['perfil-pais', codigoPais]`
- Fetch: `GET /api/paises/{codigoPais}`
- `enabled: !!codigoPais`
- `staleTime: 30 * 60 * 1000` (30 min, consistente com cache Redis)
- Retornar: `{ perfil: PerfilPais | undefined, isLoading, error }`

**`useEventosPais`:**
- Parâmetro: `codigoPais: string`
- Query key: `['eventos-pais', codigoPais]`
- Fetch: `GET /api/paises/{codigoPais}/eventos`
- `enabled: !!codigoPais`
- `staleTime: 5 * 60 * 1000` (5 min — dado mais fresco)
- Retornar: `{ eventos: EventoPais[], isLoading, error }`

### Regras de negócio aplicáveis:
- `staleTime` do perfil deve ser 30 min (alinhado ao cache Redis)
- Ao remover país, invalidar cache imediatamente

### Critérios de aceitação:
- [ ] Types TypeScript sem erros de compilação (`tsc --noEmit`)
- [ ] `useMeusPaises` retorna lista e as mutations funcionam
- [ ] `usePerfilPais` não faz fetch se `codigoPais` vazio
- [ ] `useEventosPais` com `staleTime` de 5 min

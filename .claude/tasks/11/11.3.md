# Task 11.3 – GeradorPerfilPaisService, EventosPaisService, GerarPerfilPaisJob e Command

## Objetivo
Implementar a geração de perfis via Claude API (contexto geopolítico + análise de liderança), a busca de eventos recentes por termos de busca, o Job async para geração sob demanda, o Command artisan e o agendamento no Scheduler.

## Especificação

### O que fazer:

**`GeradorPerfilPaisService`:**
```php
public function gerarPerfil(PerfilPais $pais): PerfilPais
```
- Faz 2 chamadas à Claude API separadas por `sleep(2)`:
  1. **Contexto geopolítico** (200-300 palavras): prompt pedindo análise atual do país, suas alianças, tensões e posição na ordem global
  2. **Análise de liderança** (100-150 palavras): prompt pedindo análise do líder atual, estilo de governo, posição política e impacto nas relações internacionais
- Usar `claude-sonnet-4-5` (não opus — controle de custo)
- `max_tokens: 500` para cada chamada
- Após as 2 chamadas: atualizar `contexto_geopolitico`, `analise_lideranca` e `gerado_em = now()`
- Invalidar cache Redis da chave `perfil_pais_{codigo}` após salvar

**`EventosPaisService`:**
```php
public function buscarEventosRecentes(PerfilPais $pais, int $dias = 7): Collection
```
- Busca eventos na tabela `events` onde qualquer `termos_busca` aparece no `titulo` ou `descricao` (ILIKE com OR)
- Filtrar por `created_at >= now()->subDays($dias)`
- Ordenar por `created_at DESC`
- Limitar a 10 resultados
- Retornar `Collection` de `Event`

**`GerarPerfilPaisJob`:**
- `implements ShouldQueue`
- Injetar `GeradorPerfilPaisService` via construtor
- `handle()`: chamar `gerarPerfil($pais)` — recebe `PerfilPais` $pais
- `tries = 2`, `backoff = 60`

**`GerarPerfisPaisesCommand`:**
- Signature: `paises:gerar-perfis`
- Descrição: "Gera perfis de países via IA (máx 5 por execução)"
- Lógica:
  1. Buscar 5 países com `gerado_em` mais antigo (ou null primeiro) — `orderByRaw('gerado_em IS NULL DESC, gerado_em ASC')->limit(5)`
  2. Para cada país: chamar `GeradorPerfilPaisService::gerarPerfil()` diretamente (síncrono no command)
  3. `sleep(2)` entre países
  4. Logar progresso com `$this->info()`

**Scheduler no `Kernel.php`:**
```php
$schedule->command('paises:gerar-perfis')
    ->weeklyOn(1, '03:00') // toda segunda às 03:00
    ->withoutOverlapping()
    ->runInBackground();
```

### Regras de negócio aplicáveis:
- Máximo 5 países por execução do command (controle de custo)
- Pausa de 2 segundos entre países no command
- Cache Redis `perfil_pais_{codigo}` invalidado após geração
- Usar `claude-sonnet-4-5` (não opus)

### Critérios de aceitação:
- [ ] `php artisan paises:gerar-perfis` processa exatamente 5 países
- [ ] Campos `contexto_geopolitico` e `analise_lideranca` são preenchidos com texto coerente
- [ ] `gerado_em` é atualizado após geração
- [ ] Cache Redis invalidado após geração
- [ ] Schedule registrado: `php artisan schedule:list` mostra o comando na segunda às 03:00

# Plano 11 – Perfil de País (M06)

## Objetivo
Implementar páginas de perfil por país com contexto geopolítico e análise de liderança gerados pela IA semanalmente, integradas ao Mapa de Calor (M02) e à lista "Meus Países" gerenciada pelo próprio assinante.

## Contexto de Negócio
20 países pré-cadastrados via seeder. Perfis gerados pela Claude API às segundas-feiras (máx 5 países por execução para controle de custo). Cada perfil tem 5 seções: contexto geopolítico (200-300 palavras), liderança atual (100-150 palavras), tensões ativas (do M01), indicadores econômicos (do M04) e eventos recentes (últimos 7 dias). Limite de países por plano: `essencial` → 3; `pro` → 10; `reservado/admin` → ilimitado. Cache Redis 30 minutos por perfil.

Detalhes completos em: `prd/modulo-06-perfil-pais.md`

## Escopo

### Inclui
**Backend:**
- Migrations: `perfis_paises` (codigo_pais, nome_pt, bandeira_emoji, regiao_geopolitica, contexto_geopolitico, analise_lideranca, indicadores_relevantes[], termos_busca[], gerado_em), `paises_usuarios` (user_id, codigo_pais, UNIQUE)
- `PaisesInicialSeeder` — 20 países com `termos_busca` e `indicadores_relevantes`
- Models `PerfilPais` e `PaisUsuario` com relacionamentos
- `GeradorPerfilPaisService` — 2 chamadas Claude API por país (contexto + liderança), pausa 2s entre países
- `EventosPaisService` — busca eventos via `termos_busca` em OR nas últimas 7 dias
- `GerarPerfilPaisJob` — async para geração sob demanda
- `PaisController` — `GET /api/paises`, `GET /api/paises/{codigo}`, `GET /api/paises/{codigo}/eventos`
- `PaisUsuarioController` — `GET/POST/DELETE /api/meus-paises` com verificação de limite por plano
- `AdicionarPaisRequest` — valida `codigo_pais` com `exists:perfis_paises`
- Command `paises:gerar-perfis` — processa 5 países por execução
- Scheduler: toda segunda às 03:00 com `withoutOverlapping()`
- Cache Redis 30 minutos por perfil (invalidar na regeneração)
- Exceção `LimitePaisesAtingidoException` com mensagem de upgrade

**Frontend:**
- Types para países e perfis
- Hooks: `useMeusPaises`, `usePerfilPais`, `useEventosPais` (React Query)
- `BuscaPais.tsx` — autocomplete com debounce 300ms
- `CardPais.tsx` — card na lista "Meus Países" com botão remover
- `PerfilPaisPage.tsx` — 5 seções: header (bandeira + toggle acompanhar), contexto IA, liderança IA, indicadores, eventos recentes
- `MeusPaisesPage.tsx` — lista de países seguidos + busca para adicionar
- Rotas React Router: `/paises` e `/paises/:codigo`
- Atualizar `WorldMap` do M02: clique → `/paises/{codigo}` se tem perfil, fallback para `RegionPanel`
- Adicionar "Meus Países" na navegação do dashboard

### Não inclui (fica para outro plano)
- Nada fora do escopo de países

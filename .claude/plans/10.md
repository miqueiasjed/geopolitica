# Plano 10 – Alerta Preditivo (M05)

## Objetivo
Implementar o sistema de inteligência preditiva que detecta combinações de sinais geopolíticos, gera alertas antes que eventos virem manchete e entrega notificações via dashboard (badge + painel) e e-mail (Resend) para assinantes reservados.

## Contexto de Negócio
Arquitetura em 3 camadas independentes: Camada 1 (Detecção via IA — classifica eventos em sinais military/diplomatic/supply), Camada 2 (Convergência — quando 2+ categorias convergem na mesma região), Camada 3 (Entrega — badge + e-mail via Resend). Controle de acesso por nível: `essencial` → apenas `medium`; `pro` → `medium` + `high`; `reservado/admin` → todos + recebe e-mail. Variáveis de ambiente: `ALERTA_THRESHOLD_CRITICAL=10`, `ALERTA_THRESHOLD_HIGH=7`, `ALERTA_THRESHOLD_MEDIUM=4`, `CONVERGENCIA_JANELA_HORAS=72`.

Detalhes completos em: `prd/modulo-05-alerta-preditivo.md`

## Escopo

### Inclui
**Backend:**
- Migrations: `sinais_padrao`, `alertas_preditivos`, `alertas_leituras`
- Models: `SinalPadrao`, `AlertaPreditivo`, `AlertaLeitura` com relacionamentos
- `DetectorSinaisService` — analisa eventos das últimas 48h via Claude API (lotes de 5)
- `AnalisadorConvergenciaService` — agrupa sinais por região, calcula threshold, gera análise IA
- `EntregaAlertaService` — envia e-mails em lotes de 50 via Resend (apenas `assinante_reservado` + `admin`)
- Jobs: `DetectarSinaisJob`, `AnalisarConvergenciaJob`, `EnviarEmailAlertaJob` (filas `alertas` e `emails`)
- `AlertaController` — `GET /api/alertas` (por role), `POST /api/alertas/{id}/lido`
- `AlertaPreditivoService::alertasNaoLidos()` — filtro por nível conforme plano
- Commands artisan: `alertas:detectar-sinais`, `alertas:analisar-convergencia`
- Mailable `AlertaPreditivo` + template Blade para e-mail
- Middleware `cron.secret` para rotas internas do scheduler
- Scheduler: `hourly()` para detecção, `hourlyAt(5)` para convergência
- Variáveis `.env`: `RESEND_API_KEY`, `RESEND_FROM_EMAIL`, `CRON_SECRET`, thresholds

**Frontend:**
- Types para alertas
- Hook `useAlertas.ts` com polling 5 minutos
- Hook `useMarcarAlertaLido` (mutation com invalidação de query)
- `AlertaBadge.tsx` — ícone de sino, badge vermelho pulsante com contagem
- `AlertaPanel.tsx` — painel deslizante com lista, badge por nível, botão "Marcar como lido"
- Estilos por nível: `critical` → vermelho, `high` → laranja, `medium` → amarelo
- Integrar `AlertaBadge` na `TopNav`

### Não inclui (fica para outro plano)
- Nada fora do escopo de alertas

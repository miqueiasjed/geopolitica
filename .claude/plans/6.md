# Plano 6 – Mapa de Calor Geopolítico (M02)

## Objetivo
Implementar o mapa interativo SVG do mundo onde cada país exibe intensidade de cor proporcional ao volume/gravidade de eventos geopolíticos, combinando dados do Feed (M01) com dados do GDELT Project.

## Contexto de Negócio
Intensidade combinada: `(score_m1 × 0.6) + (intensidade_gdelt × 0.4)`. O GDELT normaliza o tom negativo para escala 1–10: `round(((tom × -1) + 100) / 20) + 1`. Clique em país → painel lateral com eventos ativos. Mapa atualiza a cada 10 minutos. GDELT atualiza a cada hora (minuto :30, sem conflito com M01). Todos os assinantes têm acesso ao mapa.

Detalhes completos em: `prd/modulo-02-mapa-calor.md`

## Escopo

### Inclui
**Backend:**
- Migration `gdelt_cache` (código país, nome, total eventos, tom médio, intensidade)
- View SQL `mapa_intensidade` (combina M1 + GDELT com fórmula de peso)
- Model `GdeltCache`
- `GdeltFetcherService` — cliente GDELT API v2, normalização do tom
- `MapaIntensidadeService` — agrega dados, cache Redis 15 min
- `AtualizarGdeltJob` (fila redis, tries: 3, timeout: 120)
- `MapaIntensidadeController` — `GET /api/mapa/intensidade`
- `RegiaoEventosController` — `GET /api/mapa/regiao-eventos?regiao=`
- Scheduler: GDELT a cada hora no minuto :30

**Frontend:**
- Instalar: `react-simple-maps`, `d3-scale`, `d3-scale-chromatic`, `framer-motion`, `@radix-ui/react-tooltip`, `topojson-client`
- Baixar `public/world-110m.json` (TopoJSON Natural Earth 110m)
- Tipos `mapa.ts`
- Hooks `useMapaIntensidade.ts`, `useRegiaoEventos.ts`
- `WorldMap.tsx` — SVG com escala de cores (`#2a2a2a` → `#7f1d1d`), hover dourado, click handler
- `RegionPanel.tsx` — painel 420px animado com Framer Motion (`AnimatePresence`)
- `IntensityLegend.tsx` — legenda `bottom-6 left-6`
- Página `Mapa.tsx` — `/dashboard/mapa`
- Atualizar `TopNav.tsx` com link "Mapa de Calor"

# Plano 12 – Linha do Tempo de Crises (M07)

## Objetivo
Implementar a visualização horizontal e navegável das principais crises geopolíticas dos últimos 30 anos, combinada com eventos ativos do feed em tempo real, onde o assinante conecta padrões históricos com o presente.

## Contexto de Negócio
Duas faixas horizontais no mesmo eixo cronológico: faixa superior (crises históricas pré-cadastradas 1990–hoje, cor dourada `#C9B882`) e faixa inferior (eventos ativos do M01, cor branca `#E8E4DC`). Dados históricos são estáticos — pré-cadastrados via seeder + seed com Claude API (custo único). Clique em crise → painel de detalhe com contexto geopolítico, impacto global com métricas, impacto no Brasil. Crises históricas em cache Redis por 24 horas. Sem custo mensal adicional de IA.

Detalhes completos em: `prd/modulo-07-timeline-crises.md`

## Escopo

### Inclui
**Backend:**
- Migration `crises_historicas` (titulo, slug, ano, data_inicio, data_fim, contexto_geopolitico, impacto_global, impacto_brasil, metricas_globais[], metricas_brasil[], categorias[], content_slug nullable)
- Model `CriseHistorica` com scope por categoria
- Seeder com ~25 crises (gerado via Claude API uma única vez — incluir no código pronto para executar)
- `TimelineController` — `GET /api/timeline` (crises + eventos do M01 combinados, filtro por período)
- `TimelineDetailController` — `GET /api/timeline/crise/{slug}` (detalhe da crise)
- Rotas com cache Redis 24h para crises históricas

**Frontend:**
- Instalar `framer-motion` (se não instalado)
- Types para crises e timeline
- Hooks: `useTimeline.ts`, `useCriseDetalhe.ts`
- `TimelineBar.tsx` — SVG horizontal navegável com duas faixas, eixo de anos, scroll horizontal
- `CriseMarker.tsx` — marcador dourado clicável na faixa superior
- `EventoMarker.tsx` — marcador branco na faixa inferior (eventos do M01)
- `CriseDetailPanel.tsx` — painel deslizante (Framer Motion) com 3 seções: contexto, impacto global, impacto Brasil
- `EventoDetailPanel.tsx` — painel para eventos do feed com link para Biblioteca
- Página `Timeline.tsx` — `/dashboard/timeline`
- Atualizar `TopNav.tsx` com link "Linha do Tempo"

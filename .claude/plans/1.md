# Plano 1 – Setup e Infraestrutura Base

## Objetivo
Criar e configurar o projeto Laravel 13 + React SPA do zero, com banco de dados, Redis, filas e ambiente de desenvolvimento funcional pronto para os demais módulos.

## Contexto de Negócio
Este plano não entrega funcionalidade visível ao usuário — ele estabelece toda a fundação técnica: projeto Laravel, projeto React/Vite, conexões com MySQL e Redis, configuração do ambiente (.env), e estrutura de pastas padronizada. Sem este plano, nenhum outro pode ser executado.

## Escopo

### Inclui
- Criar projeto Laravel 13 via `composer create-project`
- Configurar `.env` (MySQL, Redis, mail, Claude API key)
- Instalar dependências: Sanctum, Spatie Permission, Anthropic SDK PHP, vknet/rss-reader
- Criar projeto React 19 + Vite + TypeScript com TailwindCSS 4, Radix UI, TanStack Query v5, React Router v7
- Configurar CORS no Laravel para aceitar requisições do frontend
- Configurar Redis como driver de cache, sessões e filas
- Criar migration base de `users` padrão Laravel + configurar Sanctum
- Configurar supervisor/Forge para queue worker
- Criar `config/claude.php` e `config/feed.php` (configurações de IA e fontes RSS)
- Verificar que `php artisan queue:work` e `php artisan schedule:run` funcionam

### Não inclui (fica para outro plano)
- Models de domínio → Plano 2 e seguintes
- Qualquer lógica de negócio → Planos 2+

## Dependências
- **Requer concluído:** Nenhum
- **Bloqueia:** Todos os demais planos

# Plano 22 – Exportação para PDF (M14)

## Objetivo
Permitir que qualquer conteúdo da plataforma — briefings, alertas preditivos, perfis de país, respostas do chat e relatório de risco — seja exportado como PDF formatado, gerado no servidor com capa, rodapé com disclaimer regulatório e, para clientes B2B, o logo da empresa substituindo o padrão.

## Contexto de Negócio
Funcionalidade de alto impacto percebido para clientes B2B: gestoras e family offices precisam apresentar análises em comitês de investimento sem fazer print de tela. O PDF é gerado on-demand (sem armazenamento), entregue como download direto e carrega a identidade visual da plataforma (ou da empresa para B2B).

Tipos exportáveis:
| Tipo | Fonte |
|---|---|
| `briefing` | tabela `conteudos` (M03) |
| `alerta` | tabela `alertas_preditivos` (M05) |
| `pais` | tabela `perfis_pais` (M06) |
| `chat` | tabela `chat_mensagens` (M09) |
| `risk_score` | dados calculados on-the-fly (M16) |

O logo B2B é buscado na tabela `licencas_b2b` via `company_slug` (M11).

## Escopo

### Inclui
- Instalação e configuração de `barryvdh/laravel-dompdf` no backend
- `ExportPdfController` com `POST /api/export-pdf` — recebe `type`, `id` e `company_slug` opcional
- `PdfTemplateService` — monta o HTML/Blade correto conforme o tipo e busca os dados
- Cinco Blade views de template PDF:
  - `pdf/layout.blade.php` — capa, header com logo (ou logo B2B), footer com disclaimer
  - `pdf/briefing.blade.php`
  - `pdf/alerta.blade.php`
  - `pdf/pais.blade.php`
  - `pdf/chat.blade.php`
- Substituto automático do logo quando `company_slug` for informado (busca `licencas_b2b`)
- Componente React `ExportPdfButton.tsx` — botão reutilizável com estado de loading e download automático via Blob
- Integração do botão nas páginas: `ConteudoLeitura.tsx` (briefing), painel de alertas, `PerfilPaisPage.tsx`, `ChatBriefings.tsx`
- Middleware de autenticação no endpoint (somente assinantes autenticados)
- Nome do arquivo: `GPI-{type}-{id8}.pdf`

### Não inclui
- Exportação de `risk_score` via PDF — implementada no Plano 24 quando o Risk Score existir
- Armazenamento dos PDFs gerados
- Histórico de exportações
- Agendamento ou envio por e-mail de PDFs

# Plano 8 – Biblioteca de Conteúdo: Frontend (M03)

## Objetivo
Construir a interface da Biblioteca — listagem com busca, filtros e scroll infinito para assinantes — e o painel admin com editor TipTap para publicação de conteúdo.

## Contexto de Negócio
O assinante navega briefings, mapas e edições de A Tese com busca por palavra-chave (debounce 500ms) e filtros combinados. O admin publica copiando texto do Claude para o editor TipTap. Cards com badge colorido por tipo: `briefing` → azul, `mapa` → amarelo, `tese` → dourado `#C9B882`. `PlanGate` exibido quando o plano é insuficiente (404 do backend). `tese_manchete` em itálico dourado no card de A Tese.

Detalhes completos em: `prd/modulo-03-biblioteca-conteudo.md` (seção 7)

## Escopo

### Inclui
- Instalar: `@tiptap/react @tiptap/pm @tiptap/starter-kit @tiptap/extension-placeholder mark.js`
- Types `biblioteca.ts`
- Hooks: `useBiblioteca.ts` (listagem infinita com cursor), `useConteudo.ts` (leitura individual)
- `ContentCard.tsx` — badge de tipo, data, região, título, tese_manchete, resumo
- `ContentReader.tsx` — renderiza HTML do TipTap com classes Tailwind Typography
- `SearchBar.tsx` — debounce 500ms
- `FilterBar.tsx` — selects de tipo, período, região + botão "Limpar filtros"
- `PlanGate.tsx` — exibido quando 404 por plano insuficiente (CTA de upgrade)
- `AdminEditor.tsx` — TipTap com barra: negrito, itálico, H2, divisor
- Página `Biblioteca.tsx` — `/dashboard/biblioteca`
- Página `ConteudoLeitura.tsx` — `/dashboard/biblioteca/:slug`
- Página `AdminBiblioteca.tsx` — `/admin/biblioteca` (listagem admin)
- Página `AdminNovoConteudo.tsx` — `/admin/novo-conteudo` (formulário de publicação)
- Atualizar `TopNav.tsx` com link "Biblioteca"

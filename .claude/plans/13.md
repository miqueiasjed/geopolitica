# Plano 13 – Radar de Eleições (M08)

## Objetivo
Implementar o calendário visual de eleições globais com painel de detalhe ao clicar, cadastradas manualmente pelo admin através do painel existente — sem IA, sem integrações externas, custo mensal zero.

## Contexto de Negócio
3 níveis de relevância por cor: Alta (vermelho `#EF4444` — muda fluxos comerciais/alianças), Média (amarelo `#FACC15` — impacto setorial), Baixa (branco 20% — contexto). O admin cadastra manualmente via painel admin (reutiliza estrutura do M03). Grade visual por mês com filtros por ano e relevância. Cache Redis 1 hora. Operação simples e barata.

Detalhes completos em: `prd/modulo-08-radar-eleicoes.md`

## Escopo

### Inclui
**Backend:**
- Migration `eleicoes` (pais, codigo_pais, data_eleicao, tipo_eleicao, relevancia, contexto_geopolitico, impacto_brasil, candidatos_principais[], content_slug nullable)
- Model `Eleicao` com scopes por ano/relevância
- `EleicaoController` — `GET /api/eleicoes` (lista por ano, filtros), `GET /api/eleicoes/{id}` (detalhe)
- `EleicaoAdminController` — CRUD completo (POST, PATCH, DELETE)
- `CriarEleicaoRequest` e `AtualizarEleicaoRequest` — validação
- Cache Redis 1 hora para lista de eleições
- Rotas admin sob middleware `role:admin`
- Seeder com ~15-20 eleições de 2026 para testes

**Frontend:**
- Types para eleições
- Hook `useEleicoes.ts` (React Query com filtro por ano)
- `RadarGrid.tsx` — grade mensal (12 colunas = 12 meses) com eleições posicionadas
- `EleicaoCard.tsx` — badge de país, data, tipo, badge de relevância colorido
- `EleicaoDetailPanel.tsx` — painel deslizante (Framer Motion) com contexto, impacto Brasil, candidatos, link para Biblioteca
- `EleicaoFilterBar.tsx` — filtro por ano (select) e relevância (toggle buttons)
- `AdminEleicaoForm.tsx` — formulário de criação/edição no painel admin
- Página `RadarEleicoes.tsx` — `/dashboard/eleicoes`
- Integrar CRUD de eleições no painel admin existente
- Atualizar `TopNav.tsx` com link "Radar de Eleições"

### Não inclui (fica para outro plano)
- Nada fora do escopo do radar de eleições
